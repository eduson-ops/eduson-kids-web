Демо · def greet()

Первая функция

  • def greet():
    Объявление. Двоеточие. Скобки пустые — параметров нет.
  • Тело с отступом

  • Вызов greet()
    Имя + скобки. Вызывать можно сколько угодно раз.

Ключевая разница

Объявление ≠ вызов

📝

Объявление

Записали рецепт. Код ждёт.

def greet():
    print("Привет!")

Вывод: ничего. Тишина.

VS
🔔

Вызов

Позвали — сделалось.

greet()
greet()

Вывод: «Привет!» × 2

«def — это "я готовлю рецепт". Вызов greet() — "испеки по рецепту"». Фраза преподавателя #2

Python

💡
Лайфхак
Если вызывать print_menu без скобок — Python скажет <function print_menu at 0x...>. Это «указатель на функцию», а не её запуск. Скобки — обязательны.

Меню видеоигры · целиком

game_menu.py 
def print_menu():
    print("1. Начать игру")
    print("2. Правила")
    print("3. Авторы")
    print("4. Выход")

def start_game():
    print("🎮 Игра начата! Удачи, герой.")

def show_rules():
    print("📜 Правило 1: ходи по карте.")
    print("📜 Правило 2: не падай в яму.")
    print("📜 Правило 3: найди сокровище.")

def show_credits():
    print("🎬 Автор: ты. Год: 2026.")

def quit_game():
    print("🚪 Пока! До встречи.")

while True:
    print_menu()
    choice = input("Выбери 1–4: ").strip()
    if choice == "1":
        start_game()
    elif choice == "2":
        show_rules()
    elif choice == "3":
        show_credits()
    elif choice == "4":
        quit_game()
        break
    else:
        print("❓ Нет такого пункта.")

Подводные камни

Что обычно ломается

🔔
Вызов без скобок
greet — это не запуск, а «указатель». Скобки () обязательны.
📐
Нет скобок в def
def greet: — SyntaxError. Пишем def greet():.
❗
Нет двоеточия
def greet() без : — traceback. Читаем вместе.
🍳
Объявил — не вызвал
Рецепт есть — блинов нет. Добавляем greet() ниже.
🪜
Тело без отступа
IndentationError. 4 пробела — вот так.
🔤
CamelCase и кириллица
Пишем show_map, не ShowMap и не показатьКарту.

Как засчитать урок

📝

Объявление без подсказки

Ребёнок пишет def имя(): с двоеточием и отступом, не подглядывая.

🔔

Вызов × N

Функция вызывается несколько раз подряд, результат виден на экране.

🧠

Понимает разницу

Объясняет, что объявление НЕ запускает код. Видит в старом коде, где нужна функция.
