Новый словарь

input()

Вопрос пользователю. Всегда возвращает строку.

🔢

int() · float()

Превращают строку в число — целое или дробное.

✨

f-строка

Магия: f"Привет, {name}!" — подставит переменную.

💬

Комментарий

# это не код — записка человеку, а не компьютеру.

↵

\n

Перенос строки внутри print().

⚙️

sep · end

Параметры print(): чем разделять и чем заканчивать.

Команда № 1

print() — сказать что-то наружу

Python
# Простой вывод print("Привет, мир!") # С переменной name = "Аня" print("Привет,", name) # С параметрами sep и end print("Python", "это", "огонь", sep=" → ", end="!\n")
sep — разделитель end — чем закончить \n — новая строка
Команда № 2

input() — задать вопрос

Python
name = input("Как тебя зовут? ") print("Привет,", name)
Запомни навсегда:
input() всегда возвращает строку.
Даже если ввёл число — это текст.
⚠️

Это НЕ сработает

age = input("Возраст? ")
print(age + 10)
# TypeError!
# str + int нельзя
Команда № 3

int() — строку превращаем в число

Матрёшка: input() внутри int().

Python
# Способ 1: сразу age = int(input("Сколько тебе лет? ")) # Способ 2: в два шага age_text = input("Сколько тебе лет? ") # строка "10" age = int(age_text) # число 10 # Теперь можно считать print("Через 10 лет тебе будет", age + 10)
int() — целое число float() — дробное (3.14)
Команда № 4 · Магия

f-строки: подставляем переменные

Python
name = "Миша" age = 12 # БЕЗ f — просто текст print("Привет, {name}!") # → Привет, {name}! # С f — подстановка! print(f"Привет, {name}!") # → Привет, Миша! # Даже арифметика внутри {} print(f"{name}, тебе {age + 10} через 10 лет")
✨

Правило

Фигурные скобки {name} — туда подставится значение. Маленькая f перед кавычкой — обязательна!

Команда № 5

Комментарии — записки будущему тебе

Python
# Это комментарий. Компьютер его не читает. # Зато поймёт любой человек (и ты через полгода). name = input("Имя: ") # спрашиваем имя age = int(input("Возраст: ")) # и возраст — сразу в число # Выводим красивую карточку print(f"Меня зовут {name}, мне {age} лет.")
Учитель: «Комментарии — это записки самому себе через полгода. Без них через месяц не поймёшь, что написал».
Живое демо

Собираем всё вместе

Python
name = input("Как тебя зовут? ") age = int(input("Сколько тебе лет? ")) print(f"Привет, {name}! Через 10 лет тебе будет {age + 10}.")
🎹

Как тебя зовут? Аня
Сколько тебе лет? 11
Привет, Аня! Через 10 лет
тебе будет 21.

Практика: «Калькулятор»

Спроси 2 числа. Выведи +, −, ×, ÷ через f-строки.

🚨

Частая ошибка

Забыть обернуть input в int(). Тогда сложение превратится в склейку строк: "2" + "3" = "23".

Python
a = int(input("Первое число: ")) b = int(input("Второе число: ")) print(f"{a} + {b} = {a + b}") print(f"{a} - {b} = {a - b}") print(f"{a} * {b} = {a * b}") print(f"{a} / {b} = {a / b}")
35–43 мин · Задание 3 · 8 минут

Практика: «Конвертер температуры»

Спроси градусы в °C — выведи в °F.

Формула

F = C × 9 / 5 + 32

float() — дробные числа
Python
c = float(input("Цельсий: ")) f = c * 9 / 5 + 32 print(f"{c}°C = {f}°F") # Бонус: округление print(f"Округлено: {round(f, 1)}°F")

⚠️ Ловушки

Частые ошибки и как их чинить

TypeError: str + int

Забыл int() вокруг input. Починка: int(input(...)).

f-строка не подставляет

Забыл букву f перед кавычкой. Проверь первый символ.

ValueError: буквы вместо цифр

Пользователь ввёл «десять» вместо 10. На уроке 9–11 научимся ловить через if.

SyntaxError: закрой кавычку

Не закрыл кавычку в input("..."). Читаем traceback вместе — номер строки.

«Куда девается введённое?»

Рисуем: твой ввод идёт в коробку-переменную. Имя коробки — name, содержимое — то, что ты ввёл.
