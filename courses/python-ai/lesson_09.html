Метафора

if / else — это как светофор

Светофор задаёт вопрос: «Можно ехать?». Ответ — True или False.
Программа делает то же самое, только вместо машин — строчки кода.

True → поехали (if) False → стоп (else)
Словарь урока

Шесть слов, которые всё меняют

Условие

Вопрос, на который есть ответ — True или False.

if

«Если» — выполни блок, когда условие True.

else

«Иначе» — выполни другое, когда False.

True / False

Булевы значения. Да или нет. Два варианта.

Операторы сравнения

== != < > <= >=

Отступ

4 пробела. Так Python понимает: блок внутри if.

Булевы значения

Весь компьютер держится на двух словах

True

Правда. Да. Условие выполнилось.

5 > 3 "python" == "python" 10 != 7
False

Ложь. Нет. Условие не сработало.

2 > 9 "cat" == "dog" 5 == "5"

Важно: True и False — с заглавной буквы. Python так хочет.

Арсенал

Шесть операторов сравнения

==Равноage == 10 # True если age=10
!=Не равноname != "Boss"
<Меньшеtemp < 10

<=Меньше или равноage <= 17

Главная ловушка

= и == — две большие разницы

=   присвоить

Python
age = 10 # положить 10 в age name = "Катя" # положить строку в name

«Коробка получает значение».

==   сравнить

Python
age == 10 # True или False? name == "Катя" # проверить равенство

«Задать вопрос: одинаковые?»

Если в условии написать if age = 10: — Python ругнётся SyntaxError.

Синтаксис

Как выглядит if

Python
if возраст >= 18: print("Можно водить")

Если условие True — Python выполняет строки с отступом. Если False — просто пропускает.

Четыре обязательные детали

  • Слово if — начинает условие
  • Условие — результат True или False
  • Двоеточие : — в конце строки
  • Отступ 4 пробела — на следующей строке
Полная форма

Добавляем else — запасной путь

Python
age = int(input("Возраст: ")) if age >= 18: print("Можно водить машину") else: print("Ещё рано")

True ветка

Выполнится код под if (с отступом).

False ветка

Выполнится код под else — ровно один раз из двух.

if без else — можно · else без if — нельзя
Правило №1

Отступ в Python — это не декорация

Правильно

Python
if x > 0: print("плюс") print("второй принт") print("это снаружи")

4 пробела = внутри if. Без отступа = снаружи.

Ошибка

Python
if x > 0: print("плюс") # IndentationError! print("таб") # не смешивай Tab и пробелы

Нет отступа — Python не понимает, где тело if.

Выбери одно: либо 4 пробела, либо Tab. Смешивать — запрещено.

1

Тёплый или холодный?

Спроси у пользователя температуру. Если меньше 10 — скажи «одевайся теплее». Иначе — «футболка ок».

Python
temp = int(input("Температура: ")) if temp < 10: print("Одевайся теплее") else: print("Футболка ок")
⏱ 8 минут
  • Не забудь int() — иначе сравнишь строку с числом
  • Двоеточие после if temp < 10
  • 4 пробела перед print
2

Угадай число

В переменной secret = 7. Спроси у игрока число. Если совпало — «молодец», иначе — «мимо».

Python
secret = 7 guess = int(input("Твоё число: ")) if guess == secret: print("Молодец!") else: print("Мимо")
⏱ 12 минут
  • Именно ==, не =
  • Поменяй secret на любое число
  • Протестируй: правильный ответ и неправильный
Бонус: добавь подсказку «больше/меньше»
3

Пароль

Пароль — "python123". Спроси у пользователя. Если совпал — «добро пожаловать», иначе — «доступ запрещён».

Python
password = "python123" entered = input("Введи пароль: ") if entered == password: print("Добро пожаловать!") else: print("Доступ запрещён")
⏱ 10 минут
  • Строки сравниваются тем же ==
  • "Python" ≠ "python" — регистр важен
  • Для строк int() не нужен
Ловушка типов

Что выведет "5" == 5?

"5" — это строка (str)

5 — это число (int)

Ответ: False

Python не переводит типы автоматически: строка не равна числу, даже если они «похожи».

Python
print("5" == 5) # False print("5" == "5") # True print(5 == 5) # True # input() всегда возвращает строку! x = input("число: ") x = int(x) # перевели в число
Мини-проект · 10 минут

Турникет метро

Python
age = int(input("Возраст: ")) if age < 7: print("Бесплатно, проходи!") else: if age <= 100: print("Пройдите, приложите карту") else: print("Вы рекордсмен!")

Три ветки

  • < 7 → бесплатно
  • 7..100 → приложи карту
  • > 100 → рекордсмен
Пока без elif — вкладываем if в else
Тренируем отступы: 4 → 8 пробелов
Частые грабли

Пять ошибок, которые ты увидишь

IndentationError

Кривой отступ — Python не понимает блок.

Фикс: 4 пробела, одинаковые везде.
if age = 10:

Один знак — это присваивание, не сравнение.

Фикс: используй ==
"5" == 5

Строка не равна числу.

Фикс: переведи через int()
else без if

else всегда относится к последнему if.

Фикс: нарисуй дерево условий.
SyntaxError: expected ':'

Забыл двоеточие после условия.

Фикс: после if ... всегда ставь :
