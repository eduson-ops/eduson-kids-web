Синтаксис

Скелет класса · разбираем на части

  • class Dog:
    Создаём чертёж. Имя — с большой буквы, PascalCase.
  • def __init__(self, ...)
    Конструктор — «момент рождения» объекта. Двойные подчёркивания — обязательны.
  • self.name = name
    Сохраняем атрибут внутри объекта.
  • def bark(self):
    Метод — действие объекта. self — первый параметр всегда.
  • rex = Dog("Рекс", 3)
    Создаём объект — вызываем класс как функцию.

Python
class Dog: def __init__(self, name, age): self.name = name self.age = age def bark(self): print(f"{self.name} говорит: Гав!") # Создаём объекты — два дома по чертежу rex = Dog("Рекс", 3) bim = Dog("Бим", 1) rex.bark() # Рекс говорит: Гав! bim.bark() # Бим говорит: Гав!
Самое важное слово

Что такое self?

«self — это слово «я» для объекта.
self.name — это «моё имя». — ключевая фраза урока

Когда Рекс лает — self = Рекс.
Когда Бим лает — self = Бим.

Python сам подставляет, кто «я» прямо сейчас. Ты не пишешь rex.bark(rex) — достаточно rex.bark().

self_demo.py
Python
class Dog: def __init__(self, name): self.name = name # "моё имя" def say(self): # self — это эта собака print(f"Я — {self.name}") rex = Dog("Рекс") bim = Dog("Бим") rex.say() # Я — Рекс (self = rex) bim.say() # Я — Бим (self = bim)
⚠️
Забыл self — получишь TypeError
Python скажет: «в методе не хватает одного параметра». Это подсказка: self.
Конструктор

__init__ — момент рождения объекта

🥚

Когда вызывается

Автоматически, как только ты пишешь rex = Dog("Рекс", 3). Ты не вызываешь его руками.

🎒

Что делает

Сохраняет в объект то, с чем он «родился»: имя, возраст, цвет. Это стартовое состояние.

⚠️

Частая ошибка

Пишут init вместо __init__. Двойные подчёркивания с двух сторон — обязательны.

init.py
Python
class Dog: def __init__(self, name, age): # это вызовется АВТОМАТИЧЕСКИ print(f"🥚 Родилась собака {name}!") self.name = name self.age = age rex = Dog("Рекс", 3) # → 🥚 Родилась собака Рекс! bim = Dog("Бим", 1) # → 🥚 Родилась собака Бим!
Два понятия · не путай

Атрибут vs Метод

🏷️ Атрибут — что у объекта ЕСТЬ

Существительное. Свойство. Данные.

self.name self.age self.color self.hp

Пример: у Рекса есть имя «Рекс». У Бима есть возраст 1.

⚡ Метод — что объект УМЕЕТ

Глагол. Действие. Функция внутри класса.

bark() run() eat() birthday()

Пример: Рекс умеет лаять. Бим умеет бежать.

Правило: атрибут — это что. Метод — это как.

Задача · класс Student

🎯 Что нужно сделать

  • Атрибуты: name, grade, scores (список)
  • Метод average() — возвращает среднюю оценку
  • Метод introduce() — «Меня зовут…, я учусь в … классе»
  • Создать 3 объекта и вывести их
💡
Подсказка
Среднее = sum(self.scores) / len(self.scores). Не забудь self. перед атрибутом.
student.py · заготовка
Python
class Student: def __init__(self, name, grade, scores): self.name = name self.grade = grade self.scores = scores def average(self): return sum(self.scores) / len(self.scores) def introduce(self): print(f"Меня зовут {self.name},") print(f"я учусь в {self.grade} классе.") misha = Student("Миша", 5, [5, 4, 5, 3]) misha.introduce() print(f"Средний балл: {misha.average()}")

Грабли · избегай

5 ошибок новичков в классах

1. Забыл self

def bark(): → TypeError. Первый параметр в методе — всегда self.

2. Нет скобок при вызове

rex.bark — это просто ссылка. Нужно rex.bark() — со скобками.

3. Одно подчёркивание

_init_ или init — не работает. Только __init__ с двойными.

4. Маленькая буква в имени класса

class dog: — так не принято. Пиши class Dog: — PascalCase.

5. Атрибут без self.

Внутри метода пишешь name = "Рекс" — это просто локальная переменная, не атрибут объекта. Нужно self.name = "Рекс".

Метод, который меняет объект

Это первый шаг к Тамагочи: покормил — сытость вверх, прошёл час — счастье вниз. — спойлер урока 32
birthday.py
Python
class Dog: def __init__(self, name, age): self.name = name self.age = age def birthday(self): self.age += 1 print(f"🎂 {self.name}, тебе теперь {self.age}!") rex = Dog("Рекс", 3) rex.birthday() # 🎂 Рекс, тебе теперь 4! rex.birthday() # 🎂 Рекс, тебе теперь 5!
