Игровая логика

Состояния клетки — всего четыре

·

Вода

Обычное поле. Сюда никто не стрелял, корабля нет.

S

Корабль

ship. Виден своему игроку, скрыт от соперника.

X

Попадание

hit. Здесь был S, игрок в него попал.

O

Промах

miss. Стрелял по воде — отметка, чтобы не повторять.

«. — вода, S — корабль, X — попадание, O — промах. Всё.»
Функция draw_board

Поле с заголовками + параметр скрытия

Параметр hide_ships=True — ты не показываешь свои корабли сопернику.

A B C D E 1 · · · · · 2 · · S · · 3 · · · O · 4 S · · · X 5 · · · · ·

Своё поле — с S. Поле соперника — без S.

draw_board.py
Python
def draw_board(board, hide_ships=True): size = len(board) letters = "ABCDEFGHIJ" # заголовок колонок print(" " + " ".join(letters[:size])) # строки с номерами for y in range(size): row = [] for cell in board[y]: if cell == "S" and hide_ships: row.append(".") else: row.append(cell) print(f"{y+1} " + " ".join(row))
Функция place_ships

Случайная расстановка через random

Алгоритм:

  1. Выбираем случайные y и x
  2. Проверяем: клетка пустая? (board[y][x] == ".")
  3. Если да — ставим S и уменьшаем счётчик
  4. Если нет — крутимся, пока не найдём пустую
⚠️
Без проверки = баг
Корабли поставятся на одну клетку. Всегда проверяй == "." перед постановкой.
place_ships.py
Python
import random def place_ships(board, count): size = len(board) placed = 0 while placed < count: y = random.randint(0, size - 1) x = random.randint(0, size - 1) if board[y][x] == ".": board[y][x] = "S" placed += 1 return board # использование: my_field = create_board(5) place_ships(my_field, 3)
Функция shoot

Выстрел → три возможных ответа

💥 hit
🌊 miss
🔁 repeat
  • S → ставим X, возвращаем "hit"
  • . → ставим O, возвращаем "miss"
  • X или O → возвращаем "repeat" (тут уже стреляли)

Главный цикл решает: при hit дать ещё ход, при miss сменить игрока, при repeat попросить другие координаты.

shoot.py
Python
def shoot(board, x, y): cell = board[y][x] if cell == "S": board[y][x] = "X" return "hit" elif cell == ".": board[y][x] = "O" return "miss" else: return "repeat"
Парсинг ввода

Как A1 превращается в (y=0, x=0)

Правило:

  • Буква → колонка → x
  • Цифра → строка → y
  • У человека счёт с 1, у компьютера — с 0
  • Всегда проверяем, что попали в поле
🔤
Секрет ord()
ord("A") = 65, ord("C") = 67. Разница — это номер колонки.
parse_coord.py
Python
def parse_coord(s, size): s = s.strip().upper() letter = s[0] num = int(s[1:]) x = ord(letter) - ord("A") y = num - 1 if 0 <= x < size and 0 <= y < size: return x, y return None # пример: parse_coord("C2", 5) # → (2, 1) parse_coord("A1", 5) # → (0, 0) parse_coord("Z9", 5) # → None

def switch_player(current): return 1 - current # трассировка: p = 0 print(switch_player(p)) # 1 p = 1 print(switch_player(p)) # 0 # в цикле: current = 0 # ... ход ... current = switch_player(current)
«Было 0, стало 1, и обратно. Одна формула — вечное чередование.»

Альтернатива с if — 3 строки. Формула — одна. Выбирай красоту.

Функция is_defeated

Проверка на победу

Идея: пройти по всем клеткам. Если нашли хоть один S — ещё не всё.

Чем короче функция, тем легче читать. Эта — идеальна: 4 строки, одна задача.

💡 Элегантнее через any

Python
def is_defeated(board): return not any("S" in row for row in board)
is_defeated.py
Python
def is_defeated(board): for row in board: for cell in row: if cell == "S": return False return True # пример: field = [[".", "X", "."], [".", ".", "O"], ["X", ".", "X"]] print(is_defeated(field)) # True — S не осталось
5 главных ошибок

На какие грабли наступают чаще всего

1 · Путают (y, x) и (x, y)

Решение: нарисовать оси на листе до кода. Запомни: board[y][x].

2 · Корабли на одной клетке

Забыли проверку. Добавь if board[y][x] == "." перед постановкой.

3 · Игра не завершается

Забыли is_defeated в цикле. Без него — бесконечная партия.

4 · Ввод A1 → ошибка

Парсим: letter = s[0].upper(); num = int(s[1:]). Проверяем границы.

5 · Функция раздулась

Правило: длиннее 15 строк — режь пополам. Одна функция — одна задача.
