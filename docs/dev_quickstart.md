# Dev Quickstart

## Один раз: установка

```bash
cd c:/Users/being/Desktop/R&D/src
npm install
```

`.npmrc` в корне форсит git bash — без этого Windows cmd.exe ломается на `&` в имени папки `R&D`.

## Запуск dev-сервера

```bash
cd apps/web
node ../../node_modules/vite/bin/vite.js
```

(или `npm run dev:web` из корня, если npm-скрипты проходят через bash)

Откроется на http://127.0.0.1:5173.

## Попробовать MVP flow

1. Главная → введи **любой 6-значный код** (напр. `123456`) или «Войти как гость».
2. Открывается редактор:
   - **Слева** — Blockly с палитрой (События / Движение / Действия / Мир / Повторы)
   - **По центру** — live Python-код (генерируется из блоков в реальном времени)
   - **Справа** — 2D игровое поле 10×10 с 🐶
3. В воркспейсе уже есть стартовая программа: «скажи Привет, повтори 4× (шаг вперёд, поворот направо)» → рисует квадрат.
4. Нажми **▶ Запустить** — при первом запуске Pyodide подгружается с CDN (~5-10 сек), потом код исполняется, команды анимируются на канвасе.
5. Изменения блоков автосохраняются в localStorage → обновление страницы возвращает прогресс.

## Что поддерживается сейчас

Блоки (9 шт):
- **События:** `при запуске`
- **Движение:** `идти вперёд N`, `повернуть налево`, `повернуть направо`, `прыгнуть`
- **Действия:** `сказать X`, `ждать N сек`
- **Мир:** `поставить блок (6 цветов)`
- **Повторы:** `повторить N раз`

Python runtime API (в Pyodide-воркере):
- `move_forward(steps)`, `turn_left()`, `turn_right()`, `jump()`
- `wait(seconds)`, `say(text)`
- `place_block(color)` — red/blue/green/yellow/purple/black

Игровой движок (2D canvas):
- Сетка 10×10, камера видит всё
- Стены по краям (персонаж говорит «Там стена!»)
- Цветные блоки под ногами персонажа
- Speech bubbles
- Плавная анимация движения + поворота

## Известные ограничения MVP-демо

|-----|--------|---------------|

## Debug-советы

**Blockly не отрисовался:** проверь что `react-router` версии 7+ и `blockly` версии 12+.

**Pyodide не запустился:** открой DevTools → Console. Первый запуск грузит с `cdn.jsdelivr.net` — должен быть доступен. Если нет — в production self-host в Yandex Object Storage.

**«Там стена!» сразу при запуске:** персонаж упёрся в край 10×10. Сделай меньше шагов или старт-позицию переопредели в `GameCanvas.initState()`.

**Python-ошибки:** показаны в оранжевой плашке под header. Text берётся из Pyodide exception.message.
