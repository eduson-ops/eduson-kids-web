Уровень 1

Автокомплит — это T9 для кода

Ты печатаешь начало. ИИ видит контекст всего файла и предлагает серым шрифтом продолжение.

  • Tab — принять подсказку
  • Esc — отклонить
  • Просто продолжай печатать — подсказка исчезнет
  • Работает в GigaIDE Desktop после входа в аккаунт Сбера
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
серое = подсказка ИИ Tab чтобы принять
Как работает автокомплит

Цикл из 4 шагов

  1. Печатаешь начало. Имя функции, пару слов комментария, первую строку — ИИ ловит идею.
  2. ИИ показывает предложение. Серым курсивом — целая строка или даже блок кода.
  3. Читаешь. Проверяешь: логика правильная? Переменные совпадают? Это главный шаг.
  4. Решаешь. Tab — принять. Esc — отказаться и продолжить самому.
⚠️
Главное правило автокомплита
«Никогда не принимай Tab не читая. Иногда ИИ подсовывает ерунду — и ты вляпаешься в баг, которого сам не написал».

Минусы и опасности

Где подвох

🤖 ИИ не думает — он угадывает

Может предложить строку, которая компилируется, но логически неверна. Проверяй запуском.

🔀 Путает переменные

Если у тебя score и scores — ИИ может подсунуть не ту. Читай глазами.

📜 Устаревший код

Иногда предлагает конструкции, которые уже никто не использует. Сверяйся с документацией.

😴 Отучает думать

Если жмёшь Tab не читая — перестаёшь понимать свой же код. Это главная ловушка.

Практика · 20 минут

Челлендж: 3 с ИИ vs 3 без

🤖 С автокомплитом

В GigaIDE. Принимай Tab, отклоняй Esc. Замеряй время секундомером.

  1. greet(name, age) — приветствие
  2. square(n) — квадрат числа
  3. count_vowels(word) — считает гласные

✍️ Руками (без ИИ)

Автокомплит выключен. Пишешь каждую букву. Тоже замеряй.

  1. say_hi(name) — просто привет
  2. is_even(n) — чётное или нет
  3. reverse_word(w) — развернуть слово
📊
После — обсуждение
Сколько времени? Где больше ошибок? Где приятнее? Вывод: ИИ — ускоритель, но ты всё равно должен понимать код.
Живой пример

count_vowels с автокомплитом

Ты печатаешь:

def count_vowels(word):

...и останавливаешься. Один Enter — ИИ видит имя функции и догадывается.

ИИ предлагает (серым):

main.py · GigaChat suggests 
def count_vowels(word):
    vowels = "аеёиоуыэюяaeiou"
    count = 0
    for letter in word.lower():
        if letter in vowels:
            count += 1
    return count

print(count_vowels("Привет"))  # 2

Читаешь. Правильно? Tab. Неправильно? Esc.

Если что-то пойдёт не так

Сложности и решения

😵 Слепо принимает все подсказки

Тренируем: сначала читаешь глазами — потом Tab. Проговариваем вслух каждую принятую строку.

🇬🇧 Cursor на английском пугает

Переключаем интерфейс в настройках. Показываем только Ctrl+L — чат по-русски работает.

🌀 Lovable генерирует не то

Переписываем промпт подробнее: добавляем секции, стиль, цвет, возраст. Итерации — норма.

⏱️ Регистрация через родителя — задержка

У преподавателя готов demo-аккаунт. Не ждём родителя — работаем с общего логина.

❓ «Зачем учиться, если ИИ всё пишет?»

Показываем намеренно плохую генерацию ИИ — чтобы увидеть ошибку, надо уметь читать код. Ты — контролёр. ИИ — подмастерье. Без тебя он ничего не сделает правильно.
