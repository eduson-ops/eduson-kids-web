05–15 · Теория-демо

Что такое vibe coding?

Работа в паре с нейросетью, где ты — архитектор, а ИИ — толковый джуниор. Он пишет черновики, ты читаешь, запускаешь и решаешь, что с этим делать.

Ты · архитектор

Ставишь задачу, читаешь код, запускаешь, ловишь ошибки, принимаешь решения. Голова — на твоих плечах.

</>

ИИ · джуниор

Пишет черновики, подсказывает синтаксис, объясняет ошибки. Быстрый, но без проверки — верить нельзя.

Vocabulary · Словарь урока

Шесть новых слов

IDE
Среда разработки — программа, где мы пишем код. Для нас это VS Code.
Interpreter · интерпретатор
Python — программа, которая читает твой код и выполняет его строчка за строчкой.
Terminal · терминал
Чёрное окно внутри VS Code, где набираешь команды и видишь ответы компьютера.
Script · скрипт
Файл с кодом, который можно запустить целиком.
.py
Расширение файла Python. Например: hello.py, game.py.
Error · ошибка
Красное сообщение в терминале — это не ругань, а письмо от компьютера: «мне что-то не нравится».

15–25 · Практика · Задание 1

Установка Python 3.11

1
10 минут

Скачать и установить

  • Открываем python.org → Downloads
  • Качаем последнюю 3.11.x (Windows installer / macOS pkg)
  • Обязательно ставим галочку «Add python.exe to PATH»
  • Install Now → ждём → Close
Проверка в терминале
PowerShell · check python
PS > python --version
Python 3.11.8
# если увидел версию — всё поставилось правильно
!

Галочка «Add python.exe to PATH»

Без неё в терминале будет ошибка «python не является внутренней командой». Если забыл — переустановить Python с правильной галочкой.

Troubleshooting · Типичные сложности

Пять граблей, на которые все наступают

×

'python' не является внутренней командой

Забыл галочку PATH при установке. Решение: переустановить Python с правильной галочкой.

×

Терминал не открывается в VS Code

Ярлык Ctrl+` (клавиша под Esc, левый верхний угол клавиатуры).

×

Путаница " и ' в строке

Любые кавычки работают, главное — парные. Нельзя открыть " и закрыть '.

×

Родитель сам всё делает

Мягко: «пусть Дима нажмёт — я для этого здесь». Руки ученика — руки вайб-кодера.

×

Ребёнок стесняется ошибок

«ошибка — это не плохо, это просто письмо от компьютера». Установи норму: ошибки — часть процесса.
