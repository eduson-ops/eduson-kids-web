03 · Как объяснить JSON ребёнку

JSON — это вложенные папки

Ключ — это путь, куда идти. Значение — это что там лежит. Ключ может указывать на другое оглавление — это и есть вложенность.

«JSON — как оглавление в книжке. Ключ → куда идти, значение → что там лежит. Ключ может указывать на другое оглавление». — Ключевая фраза преподавателя

Словарь урока

JSON словарь / dict список / list ключ (key) значение (value) вложенность парсинг

04 · Что такое JSON

Универсальный язык интернета

Любой API в мире — от погоды до игр — возвращает данные в формате JSON.

{ }

Объект

Фигурные скобки. Внутри пары «ключ: значение». Как словарь в Python.

{"name": "Дима", "age": 12}
[ ]

Массив

Квадратные скобки. Список значений через запятую. Как list в Python.

["USD", "EUR", "CNY"]
" "

Значения

Строки в кавычках, числа без кавычек, true/false, null.

"строка", 42, true, null

Главное превращение

1
Сервер
присылает текст JSON
→
2
r.json()
распаковывает в Python
→
3
dict
работаем ключами

05 · JSON в действии

Открываем JSON ЦБ РФ в браузере

json_pretty.py
import requests
import json

r = requests.get("https://www.cbr-xml-daily.ru/daily_json.js")
data = r.json()

# Красивый вывод — с отступами и кириллицей
pretty = json.dumps(data, indent=2, ensure_ascii=False)
print(pretty[:500])

Что делает json.dumps

  • indent=2 — отступы для читабельности
  • ensure_ascii=False — не ломает русский
  • Возвращает строку, можно печатать или сохранять в файл

Подводный камень

Без ensure_ascii=False увидишь:
"\u0434\u043e\u043b\u043b\u0430\u0440" вместо «доллар».

Извлекаем вложенные ключи
за один проход

extract_values.py
import requests

r = requests.get("https://www.cbr-xml-daily.ru/daily_json.js")
data = r.json()           # ← превращаем JSON в словарь Python

# Достаём курсы через цепочку ключей
eur = data["Valute"]["EUR"]["Value"]
hkd = data["Valute"]["HKD"]["Value"]

print(f"Евро: {eur:.2f} ₽")
print(f"Гонконгский доллар: {hkd:.2f} ₽")
r.json()

Вызывается у объекта ответа (response). Возвращает готовый dict.

["Valute"]["EUR"]

Два захода в словарь: сначала в общий раздел, потом в конкретную валюту.

:.2f

Формат f-строки: 2 знака после запятой. Из 92.4537 → 92.45.

Как сделать таблицу красивой

Секрет — мини-язык внутри {...}. После двоеточия пишем: сторона, ширина, формат.

ЗаписьЧто делаетПример
{x:<10}влево, ширина 10USD       
{x:>10}вправо, ширина 10       USD
{x:^10}по центру   USD    
{x:.2f}2 знака после точки92.45
{x:>8.2f}вправо, ширина 8, 2 знака   92.45

Запомни

  • < — влево, для слов
  • > — вправо, для чисел
  • ^ — центр, для заголовков
  • число — ширина колонки в символах
  • .2f — дробное, 2 знака
«Ряды совпадают? Значит не соврал в ширине». — Проверь на глаз

11 · Финал урока

«Финансовая сводка на утро»

6 валют, отсортированы по курсу от большего к меньшему.

currency_table.py
import requests

r = requests.get("https://www.cbr-xml-daily.ru/daily_json.js")
data = r.json()
valute = data["Valute"]

codes = ["USD", "EUR", "GBP", "CNY", "HKD", "JPY"]

# Сортируем от самого дорогого к самому дешёвому
codes_sorted = sorted(
    codes,
    key=lambda c: valute[c]["Value"],
    reverse=True
)

print(f"{'Код':<6}| {'Название':<28}| {'Курс ₽':>10}")
print("-" * 50)
for code in codes_sorted:
    v = valute[code]
    print(f"{code:<6}| {v['Name']:<28}| {v['Value']:>10.2f}")

Как работает lambda

«Возьми код валюты c, посмотри её Value, отсортируй по нему».

reverse=True — от большего к меньшему.

Отправляем папе

Скриншот в TG с подписью «пап, смотри мою первую финансовую сводку».

Словарь vs Список
12 · Разница двух структур

Словарь или список?
Смотри на скобки.

Словарь  { } 

Доступ по ключу-строке

valute = {
  "USD": 92.45,
  "EUR": 100.73
}

valute["USD"]  # 92.45

Используй, когда имена важны и порядок нет.

Список  [ ] 

Доступ по номеру-индексу

codes = [
  "USD",
  "EUR"
]

codes[0]  # "USD"
codes[1]  # "EUR"

Используй, когда порядок важен, а имён нет.

Частая ошибка

Ребёнок пишет data[0], а нужно data["Valute"]. Python говорит: «а у тебя тут словарь — не знаю я ключа 0». Посмотри на скобки в JSON-файле: фигурные — словарь, квадратные — список.

13 · Типичные грабли

5 ошибок, которые встретишь

KeyError: 'USD'

Регистр или опечатка

Python не нашёл ключ. Смотри точное написание в cbr.json. "usd" ≠ "USD".

TypeError: list indices must be int

Путаешь список и словарь

Пишешь data[0] вместо data["ключ"]. Посмотри на скобки исходника.

Длинная строка вместо JSON

Нет форматирования

Shift+Alt+F в VS Code или плагин JSON Viewer.

f-строка ломается

Путаешь кавычки

Если f-строка в "...", внутри используй '...': f"{v['Name']}".

AttributeError: dict no .json

.json() — только у response

Правильно: r.json(). Неправильно: data.json().

UnicodeDecodeError

Кодировка

При сохранении — encoding="utf-8". При json.dumps — ensure_ascii=False.

17 · Если осталось время

История курсов в history.json

Каждый запуск дописывает в файл запись «дата → курсы». Через неделю — график движения валют.

history_saver.py
import requests, json
from datetime import datetime

r = requests.get("https://www.cbr-xml-daily.ru/daily_json.js")
data = r.json()
valute = data["Valute"]

today = datetime.now().strftime("%Y-%m-%d")
snapshot = {
    code: valute[code]["Value"]
    for code in ["USD", "EUR", "CNY"]
}

# Читаем старое или начинаем с нуля
try:
    with open("history.json", "r", encoding="utf-8") as f:
        history = json.load(f)
except FileNotFoundError:
    history = {}

history[today] = snapshot

with open("history.json", "w", encoding="utf-8") as f:
    json.dump(history, f, indent=2, ensure_ascii=False)

print(f"Сохранено на {today}: {snapshot}")

Что нового

  • json.load(f) — читаем JSON из файла
  • json.dump(obj, f) — пишем JSON в файл
  • datetime.now().strftime(...) — сегодняшняя дата
  • try/except FileNotFoundError — первый запуск

Зачем

Так работает большинство финансовых приложений: раз в день снимают данные и копят историю.
