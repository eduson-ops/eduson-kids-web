Цикл for
Известное количество

Когда знаешь, сколько раз

for проходит по последовательности (числа, список, строка) и выполняет блок для каждого элемента.

Синтаксис
for <переменная> in <последовательность>:
    # тело цикла — с отступом
  • range(5) → 0, 1, 2, 3, 4
  • range(1, 11) → 1..10 (правая граница не включается!)
  • range(0, 10, 2) → 0, 2, 4, 6, 8

Цикл while
Пока условие истинно

Когда знаешь «пока что»

while проверяет условие перед каждой итерацией. Пока True — работает. Стало False — выходит.

Синтаксис
while <условие>:
    # тело — выполняется пока True
⚠️

Бесконечный цикл? Если забыл поменять условие внутри — программа зависнет. Стоп-кран: Ctrl + C

Главный выбор

Как выбрать правильный цикл

Главный вопрос: я знаю заранее, сколько раз повторять?

FOR

Известное количество шагов

Есть конкретная последовательность или число повторов.

  • Таблица умножения 1..10
  • Перебор списка друзей
  • Печать 100 звёздочек
  • Обход символов в строке
Python
for i in range(1, 11): print(f"7 × {i} = {7*i}")
WHILE

Неизвестное количество

Пока выполняется условие — работаем. Когда остановится — не знаем.

  • Пока пользователь не угадал
  • Пока ответ сервера ≠ 200
  • Пока не ввели «стоп»
  • Пока вода не закипит
Python
secret = 42 while guess != secret: guess = int(input("? "))

Игра «Угадай число»

Python загадывает число от 1 до 100. Ты угадываешь. Он подсказывает «больше» или «меньше».

01

import random

Стандартная библиотека. Даёт случайные числа. random.randint(1, 100) — случайное целое от 1 до 100 включительно.

02

while True

Бесконечный цикл. Пока не угадали — продолжаем. Когда угадали — break.

03

Сравнение + подсказка

Внутри цикла: спросили число, сравнили с секретом. Больше / меньше / точно.

04

Счётчик попыток

Переменная tries = 0, внутри цикла tries += 1. В конце — сколько раз пытался.

05

Ограничение попыток

Если tries == 10 — выход по break с сообщением: «Загадано было N».

06

Поздравление

После цикла — финальный print с количеством попыток и победной фразой.

guess_game.py
Полный листинг

Код игры

Всего 14 строк — и у тебя рабочая игра с 10 попытками.

  • randint(1, 100) — случайное число
  • while tries < 10 — пока попытки есть
  • guess = int(input(...)) — не забыть int!
  • break — выход при угадывании

💡 tries += 1 — это короткая запись tries = tries + 1

  guess_game.py
Python
import random secret = random.randint(1, 100) tries = 0 print("Я загадал число от 1 до 100. У тебя 10 попыток!") while tries < 10: guess = int(input("Твой вариант: ")) tries += 1 if guess == secret: print(f"🎉 Угадал за {tries} попыток!") break elif guess < secret: print("Больше ⬆️") else: print("Меньше ⬇️") else: print(f"Попытки закончились. Загадано было {secret}")

Q1

Что такое итерация?

Один проход цикла. Если range(5) — значит 5 итераций.

Почему мы использовали while, а не for?

Потому что мы не знали заранее, сколько попыток сделает пользователь. Могло быть 2, могло быть 10.
