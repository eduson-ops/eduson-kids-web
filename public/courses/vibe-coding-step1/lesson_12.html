Задание 1 · 10 минут

Приветствие и три вопроса дня

В main.py собираем цикл на 3 итерации. Каждый вопрос — ответ пользователя — сохранение в history.json через функцию storage.save.

main.py · три вопроса дня 
from datetime import datetime
from gigachat_client import ask
from storage import save, load_all, count

def main():
    print(f"Привет! У нас уже {count()} записей в дневнике.")
    print("Расскажи, как прошёл день — задам 3 вопроса.\n")

    questions = [
        "Что ты сегодня делал?",
        "Что тебя порадовало?",
        "Чего ты хочешь на ужин?",
    ]

    for q in questions:
        answer = input(f"{q}\n> ")
        save({
            "q": q,
            "a": answer,
            "time": datetime.now().strftime("%d.%m.%Y %H:%M"),
        })
        print("Записал!\n")

if __name__ == "__main__":
    main()
Задание 2 · 10 минут

Пересказ через GigaChat — главная фишка

Забираем последние 3 записи из history.json, упаковываем в промпт и просим GigaChat пересказать с юмором — как бабушка.

main.py · финальный пересказ 
# после 3 вопросов — берём последние записи
last_three = load_all()[-3:]

# склеиваем в текст для промпта
story = "\n".join(
    f"- {item['q']} Ответ: {item['a']}"
    for item in last_three
)

system = (
    "Ты добрая бабушка. Пересскажи кратко и с юмором, "
    "что рассказал тебе внук. 3-4 предложения."
)

retell = ask(f"{system}\n\nВот что он рассказал:\n{story}")

print("\n=== СВОДКА ДНЯ ОТ БАБУШКИ ===")
print(retell)
Задание 3 · 10 минут

Прогон три раза — ловим баги

Защищаем от падений

  • Пустой history.json → JSONDecodeError → возвращаем пустой список.
  • Сеть отвалилась → try/except вокруг ask(), заглушка «не смог».
  • Пользователь ввёл пустую строку → перепросим.
  • GigaChat вернул много текста → ограничиваем max_tokens=512.
storage.py · защитный load_all 
import json, os

def load_all() -> list:
    if not os.path.exists("history.json"):
        return []
    try:
        with open("history.json", "r",
                  encoding="utf-8") as f:
            return json.load(f)
    except json.JSONDecodeError:
        return []
