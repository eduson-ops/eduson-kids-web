HTTP · как это устроено

Запрос — ответ: диалог клиента и сервера

Клиент (ты)

Python-скрипт

Составляет URL, открывает соединение и отправляет запрос. Ждёт ответа до нескольких секунд.

requests.get(url)
GET
JSON
Сервер (ЦБ РФ)

cbr-xml-daily.ru

Принимает запрос, смотрит текущие курсы валют и отправляет обратно JSON с данными.

{ "Valute": { "USD": {...} } }

URL

Адрес сервера. Начинается с https://. В нашем случае — ЦБ РФ.

Метод GET

«Я только посмотрю». Не изменяет данные на сервере, только читает.

Статус-код

Трёхзначное число в ответе. 200 — всё хорошо.

Статус-коды HTTP

Что означают цифры в ответе

Статус-код — первое, что проверяет любой программист. Он говорит коротко: всё хорошо или что-то пошло не так.

2xxУспех

200 OK — ответ пришёл, читай данные.

3xxПеренаправление

301/302 — сервер отправляет тебя в другое место.

4xxОшибка клиента

404 не найдено, 403 запрещено, 401 нет доступа.

5xxОшибка сервера

500 — сервер сам упал. Ты не виноват, просто подожди.

Проверяем всегда

r = requests.get(url)
if r.status_code == 200:
    print("Ответ получен!")
else:
    print("Что-то пошло не так:", r.status_code)

Что видит ученик сегодня

>>> print(r.status_code)
200
>>> _

Первый GET-запрос

Код ученика

import requests

url = "https://www.cbr-xml-daily.ru/daily_json.js"
r = requests.get(url)

print("Статус:", r.status_code)
print("Первые 200 символов ответа:")
print(r.text[:200])
✓ статус 200 видно начало JSON

Что важно объяснить

  • r — объект ответа. В нём лежат все данные, которые прислал сервер.
  • r.status_code — число-код статуса.
  • r.text — весь ответ как строка.
  • [:200] — срез, берём первые 200 символов (чтобы не захламлять экран).

Достаём курс доллара

Код шаг за шагом

import requests

url = "https://www.cbr-xml-daily.ru/daily_json.js"
r = requests.get(url)

# Превращаем ответ в структуру Python
data = r.json()

# Лезем по цепочке ключей: Valute -> USD -> Value
usd = data["Valute"]["USD"]["Value"]

# Красиво выводим
print(f"Доллар сегодня: {usd:.2f} рубля")

Цепочка ключей

data→ "Valute"→ "USD"→ "Value"

Каждый ключ — следующая «папка». Внутри последней лежит число — курс.

Мини-проект · 45:00 — 55:00

exchange.py — курс с динамикой

Сравниваем сегодняшний курс со вчерашним. Показываем стрелку роста или падения.

Полный код проекта

import requests

URL = "https://www.cbr-xml-daily.ru/daily_json.js"

r = requests.get(URL)
data = r.json()

usd = data["Valute"]["USD"]
today = usd["Value"]
yesterday = usd["Previous"]
diff = today - yesterday

date = data["Date"][:10]  # YYYY-MM-DD

if diff > 0:
    sign = "▲"
elif diff < 0:
    sign = "▼"
else:
    sign = "="

print(f"Курс доллара на {date}: {today:.2f} руб.")
print(f"{sign} {diff:+.2f} за сутки")

Если что-то пошло не так

Типичные ошибки и как их решать

ModuleNotFoundError

«No module named 'requests'»

Забыл поставить библиотеку. Решение: pip install requests или python -m pip install requests.

pip is not recognized

«pip не является командой»

pip не попал в PATH. Решение: python -m pip install requests. Почти всегда работает.

ConnectionError

Нет интернета / блокировка

Сначала проверяем, открывается ли в браузере любой сайт. Если школьный Wi-Fi режет — переключаемся на раздачу с телефона.

KeyError: 'USD'

Ключ не найден

Проверить регистр. В ЦБ всё с большой: "Valute", "USD", "Value".

403 / 404

Опечатка в URL

Сверить по памятке. Самая частая ошибка — дефис не там или .ru / .com.

Пустой ответ

Забыл сохранить в переменную

Фраза: «сохрани в r, иначе не сможешь посмотреть ответ». requests.get(url) без присваивания — ответ уходит в пустоту.

USDEURJPYCNYGBP

import requests

r = requests.get("https://www.cbr-xml-daily.ru/daily_json.js")

Бонус для опережающих

Конвертер из командной строки

Запуск вида python exchange.py 100 USD RUB — выводит, сколько рублей в 100 долларах.

Идея

  • Читаем аргументы командной строки через sys.argv
  • Парсим: сумма, откуда, куда
  • Берём курс из ЦБ РФ и переводим
  • Показываем результат
import sys, requests

amount = float(sys.argv[1])
src = sys.argv[2]
dst = sys.argv[3]

r = requests.get("https://www.cbr-xml-daily.ru/daily_json.js")
data = r.json()

# переводим через рубль
rub = amount * data["Valute"][src]["Value"] if src != "RUB" else amount
result = rub / data["Valute"][dst]["Value"] if dst != "RUB" else rub

print(f"{amount} {src} = {result:.2f} {dst}")
