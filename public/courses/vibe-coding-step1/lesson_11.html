Демо · код

gigachat_client.py — разговорчивый холодильник

assistant/gigachat_client.py
Python
# gigachat_client.py — общается с нейросетью import os import requests from dotenv import load_dotenv load_dotenv() # подтягиваем .env в окружение TOKEN = os.getenv("GIGACHAT_TOKEN") URL = "https://gigachat.devices.sberbank.ru/api/v1/chat/completions" def ask(question: str) -> str: """Отправляет вопрос GigaChat и возвращает ответ строкой.""" headers = {"Authorization": f"Bearer {TOKEN}"} data = { "model": "GigaChat", "messages": [{"role": "user", "content": question}], "temperature": 0.7, } r = requests.post(URL, json=data, headers=headers, verify=False) return r.json()["choices"][0]["message"]["content"]

Снаружи видна только ask(). Всё остальное — внутренние детали. Это и есть инкапсуляция.

Демо · код

storage.py — аккуратная кладовка

assistant/storage.py
Python
# storage.py — хранит историю диалогов в history.json import json from pathlib import Path FILE = Path("history.json") def load_all() -> list: """Возвращает список всех записей. Если файла нет — пустой список.""" if not FILE.exists(): return [] return json.loads(FILE.read_text(encoding="utf-8")) def save(entry: dict) -> None: """Добавляет одну запись {"q":..., "a":..., "time":...} в историю.""" data = load_all() data.append(entry) FILE.write_text(json.dumps(data, ensure_ascii=False, indent=2), encoding="utf-8")

main.py — дирижёр оркестра

assistant/main.py
Python
# main.py — точка входа проекта from datetime import datetime from gigachat_client import ask # наш модуль from storage import save # наш модуль def ask_user() -> str: return input("Твой вопрос: ") def main(): question = ask_user() answer = ask(question) print("Ассистент:", answer) save({ "q": question, "a": answer, "time": datetime.now().isoformat(timespec="seconds"), }) if __name__ == "__main__": main()

Создай каркас в VS Code

Открой VS Code → Ctrl+K Ctrl+O → выбери папку проекта. Создай структуру руками: ПКМ по проводнику → New Folder, потом New File.

my_project/ assistant/ main.py gigachat_client.py storage.py .env # history.json не создавай руками!

Правило

history.json создастся сам при первом сохранении. Если создать пустой — получишь ошибку JSONDecodeError.

Если застрял — подсказка
В терминале VS Code: mkdir assistant && cd assistant, потом type nul > main.py (Windows) или touch main.py (Mac/Linux).

ШАГ 3 / 5 ~ 12 мин

Напиши gigachat_client.py

Функция одна: def ask(question: str) -> str. Берёт вопрос, возвращает ответ строкой. Без print внутри — модуль не должен шуметь.

assistant/gigachat_client.py
Python
import os, requests from dotenv import load_dotenv load_dotenv() TOKEN = os.getenv("GIGACHAT_TOKEN") URL = "https://gigachat.devices.sberbank.ru/api/v1/chat/completions" def ask(question: str) -> str: headers = {"Authorization": f"Bearer {TOKEN}"} data = {"model": "GigaChat", "messages": [{"role": "user", "content": question}], "temperature": 0.7} r = requests.post(URL, json=data, headers=headers, verify=False) return r.json()["choices"][0]["message"]["content"]
Если застрял — подсказка
Нет requests? В терминале: pip install requests python-dotenv. Функция не видит токен? Проверь, что .env в той же папке, где запускаешь.
ШАГ 4 / 5 ~ 15 мин

Напиши storage.py

Две функции: save(entry: dict) — добавить запись, load_all() -> list — прочитать все. Хранит в history.json.

assistant/storage.py
Python
import json from pathlib import Path FILE = Path("history.json") def load_all() -> list: if not FILE.exists(): return [] return json.loads(FILE.read_text(encoding="utf-8")) def save(entry: dict) -> None: data = load_all() data.append(entry) FILE.write_text( json.dumps(data, ensure_ascii=False, indent=2), encoding="utf-8", )
Если застрял — подсказка
Файл пропадает? Ты открыл его в режиме "w" — он перезаписывает. Наш код использует write_text после load_all, так что записи накапливаются.
ШАГ 5 / 5 ~ 10 мин

Собери main.py и запусти

Точка входа импортирует обе твои функции — ask и save. Спрашивает пользователя, получает ответ, печатает, сохраняет.

assistant/main.py
Python
from datetime import datetime from gigachat_client import ask from storage import save def ask_user() -> str: return input("Твой вопрос: ") def main(): q = ask_user() a = ask(q) print("Ассистент:", a) save({"q": q, "a": a, "time": datetime.now().isoformat(timespec="seconds")}) if __name__ == "__main__": main()

Если что-то пошло не так

Типичные сложности и как их победить

ModuleNotFoundError

gigachat_client не находится. Ты запускаешь не из папки проекта. Делай cd assistant, потом python main.py.

Циклические импорты

Редко, но бывает. Правило: если A импортит B — B не должен импортить A.

.env не видно

Не установлен python-dotenv, или забыл load_dotenv() в начале файла.

history.json не растёт

Проверь режим открытия. "w" перезаписывает, мы используем load → append → write.

Бонус · для опережающих

Добавь цвет в консоль

Библиотека rich превращает скучный белый текст в цветной и стильный. Установи и замени print на rich.print.

pip install rich

from rich import print print("[bold cyan]Ассистент:[/] ", end="") print(f"[green]{answer}[/]") print(f"[dim]сохранено {datetime.now():%H:%M:%S}[/]")
