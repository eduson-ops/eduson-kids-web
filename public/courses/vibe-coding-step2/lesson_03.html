05 / 22FastAPI vs Flask

Почему FastAPI, а не Flask в 2026

FastAPI современный

  • Async из коробки — не заикается на долгих запросах
  • Swagger и Redoc — бесплатно, без единой строки
  • Pydantic для валидации — данные проверяются автоматически
  • Самый популярный бэкенд в русских AI-стартапах

Flask старая школа

  • Синхронный — блокирует на LLM-запросах
  • Swagger — ставить отдельные плагины и ждать сюрпризов
  • Валидация — писать руками или плагин
  • Живой, но для новых проектов — нет причин брать

Постарайся не ставить Flask в 2026.

08 / 22Шаг 1 — venv

Виртуальное окружение: свой Python для проекта

Без venv ты ставишь пакеты «в общак». Один проект хочет FastAPI 0.110, другой — 0.95. Конфликт. Проект ломается.

venv решает это: у каждого проекта свой Python со своими пакетами.

  • Создание: python -m venv .venv
  • Активация Windows: .venv\Scripts\activate
  • Активация macOS/Linux: source .venv/bin/activate
  • В терминале появится префикс (.venv) — ты внутри
# создаём папку проекта
mkdir api && cd api

# создаём виртуальное окружение
python -m venv .venv

# активируем (Windows)
.venv\Scripts\activate

# активируем (macOS / Linux)
source .venv/bin/activate

# проверяем
which python
# .../.venv/bin/python — значит, всё ок
bash
09 / 22Шаг 2 — pip install

Ставим FastAPI + uvicorn

Когда venv активирован, всё, что ты ставишь через pip, попадает только в этот venv.

  • fastapi — сам фреймворк
  • uvicorn — ASGI-сервер, чтобы всё это запустить

Pip скачает несколько зависимостей — starlette, pydantic, click и т.д. Это нормально, они все нужны.

# внутри активированного venv
pip install fastapi uvicorn

# проверка, что поставилось
pip list | grep -i fastapi
# fastapi    0.115.x
bash
10 / 22Шаг 3 — main.py

Твой первый сервер в 12 строках

Python
from fastapi import FastAPI from datetime import datetime app = FastAPI(title="Vibe Step 2 API") @app.get("/hello") def hello(): return {"message": "hello, vibe"} @app.get("/ping") def ping(): return {"pong": True} @app.get("/time") def current_time(): return {"now": datetime.now().isoformat()}

Три декоратора @app.get() — три эндпоинта. FastAPI всё остальное сделает сам: маршрутизацию, сериализацию в JSON, валидацию, документацию.

11 / 22Шаг 4 — запускаем

uvicorn --reload

uvicorn main:app --reload

# что это значит:
#   main   = имя файла main.py
#   app    = объект FastAPI внутри
#   --reload = перезапуск при изменениях
bash

В терминале увидишь:

INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
bash

--reload — что это

Ты меняешь код и сохраняешь → uvicorn сам перезапускается. Не нужно Ctrl+C → стрелка вверх → Enter. Must-have в разработке.

Но не в проде

На боевом сервере --reload выключают — там код не меняется налету, а лишний watcher ест ресурсы.

В Flask это 2–3 доп-пакета и день настройки. В FastAPI — из коробки.

15 / 22Шаг 8 — .gitignore

Что НЕ коммитим

Папка .venv/ — это твои скачанные пакеты. Они огромные, и у каждого свои. Её НЕ коммитят, а восстанавливают локально из requirements.txt.

Добавь в .gitignore:

.venv/
__pycache__/
*.pyc
.env
.DS_Store
bash

Золотое правило

Коммит должен содержать инструкцию, как собрать проект, а не готовую сборку.

Чужой человек с твоего README должен:

  1. Клонировать репо
  2. Сделать venv
  3. pip install -r requirements.txt
  4. Запустить uvicorn
16 / 22Типовые косяки

Три ошибки, которые случатся у тебя

01

Порт 8000 занят

Ошибка: [Errno 48] Address already in use

→ uvicorn main:app --port 8001

02

ModuleNotFoundError: fastapi

Забыл активировать venv.

→ Проверь префикс (.venv) в терминале. Активируй заново.

03

--reload не срабатывает

На Windows uvicorn не всегда видит изменения.

→ Рестарт вручную: Ctrl+C и заново.

Если сервер не запускается — 90% проблем на первой строке ошибки. Читай сверху вниз.

19 / 22Домашка · 20 минут

ДЗ: /random_number

Добавь четвёртый эндпоинт с query-параметром:

  • Метод: GET
  • Путь: /random_number
  • Query: ?max=100 (дефолт)
  • Ответ: случайное число от 0 до max
  • Коммит с сообщением feat: random number endpoint

Подсказки: import random, random.randint(0, max), query-параметры в FastAPI — это аргументы функции с дефолтным значением.

Python
from fastapi import FastAPI import random app = FastAPI() @app.get("/random_number") def random_number(max: int = 100): return { "value": random.randint(0, max), "max": max, }

Проверь в Bruno: http://localhost:8000/random_number?max=1000

21 / 22Бонус + деньги

Для опережающих · + как это продаётся

БОНУС

POST /echo

Эндпоинт принимает JSON-тело через Pydantic-модель и возвращает его обратно с полем received_at.

from pydantic import BaseModel
from datetime import datetime

class EchoIn(BaseModel):
    message: str

@app.post("/echo")
def echo(body: EchoIn):
    return {
      "you_sent": body.message,
      "received_at": datetime.now().isoformat(),
    }
