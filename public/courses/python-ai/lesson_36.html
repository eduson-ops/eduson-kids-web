Сложности и решения

Что обычно ломается

🇬🇧
Lovable пишет по-английски
В промпте обязательно: «на русском языке».
🤷
Шаблонное «я люблю программирование»
Пиши конкретно: «написал Тамагочи, гордился, что у кота появились эмоции».
🖼️
Картинка Kandinsky не грузится
На Яндекс.Диск → прямая ссылка → вставить URL в Lovable.
🔠
Шрифт крив — мелко/крупно
Отдельный промпт: «сделай заголовок 48px, текст — 18px».
✉️
Форма контакта не работает
Lovable не настраивает бэкенд. Форма — «для красоты». Реально заведём в курсе Vibe Coding.
🎨
Цвета «сиреневенький»
Только #HEX. «Сиреневенький» ИИ понимает как угодно.

Мини-игра прямо на сайте

Одним промптом превращаем секцию в живую игру. Lovable встроит JavaScript прямо в страницу.

Промпт

«добавь встроенную мини-игру «Виселица» на JavaScript во вторую секцию»

Игра загадывает слово, показывает прочерки, ты кликаешь буквы, за неверные — часть виселицы. Прямо в браузере.

game.js · сгенерировано Lovable
JavaScript
const words = ["python", "нейросеть", "промпт", "виселица"]; const word = words[Math.floor(Math.random() * words.length)]; let guessed = Array(word.length).fill("_"); let lives = 6; function onLetterClick(letter) { if (word.includes(letter)) { for (let i = 0; i < word.length; i++) { if (word[i] === letter) guessed[i] = letter; } } else { lives--; } render(); } function render() { document.getElementById("w").textContent = guessed.join(" "); document.getElementById("l").textContent = "❤ " + lives; }

3 · Защита 3 минуты

Домашнее задание · ≤20 минут

ДЗ · собираем отзывы

  • Отправь ссылку родителям
    В семейный чат: «Смотрите мой сайт: bla.lovable.app».
  • Собери 3 отзыва
    Мама, папа, бабушка/друг. Что понравилось? Что улучшить?
  • Внеси одно улучшение
    Новый проект в секции, цвет кнопки, опечатку — одним промптом в Lovable.
  • Сохрани скриншот
    До и после — чтобы показать в школе, что ты итерируешь.
