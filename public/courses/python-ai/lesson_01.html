Что такое Python?

Язык, на котором мы разговариваем с компьютером

🐍

Назван не в честь змеи

А в честь британской комик-группы «Летающий цирк Монти Пайтона». Создатель Python Гвидо ван Россум был их фанатом.

💬

Как ты говоришь с компьютером

Ты пишешь команду на Python — компьютер её читает построчно и выполняет. Как если бы ты диктовал инструкцию роботу.

🚀

Где живёт Python

В Instagram, YouTube, Spotify, нейросетях, играх, научных лабораториях. Даже Google его использует.

✨

Почему его любят

Код читается почти как английский. Никаких заумных скобочек. Идеально для первого языка программирования.

Как код превращается в результат

Ты пишешь → интерпретатор переводит → компьютер выполняет

Важно: интерпретатор работает построчно. Если на 3-й строке ошибка — строки 1 и 2 уже выполнились. Это значит: ошибки искать просто.

Что такое IDE?

IDE — Integrated Development Environment. Редактор с суперсилами.

Простая метафора: IDE — это кухня повара. Всё под рукой: ножи, плита, доска. Без IDE писать код — это как готовить на костре в лесу. Можно, но зачем?

Первая программа

Это та самая магическая строчка — print("Hello, world")

Python
# Файл hello.py print("Hello, world")
$ python hello.py
Hello, world
  • print — это команда «выведи на экран»
  • ( ) — скобки говорят, что это функция, а внутри её «еда»
  • " " — кавычки обозначают строку, текст для компьютера

Один print — тысяча вариантов

Меняй текст внутри кавычек — результат меняется

Python
# Русский текст print("Привет, я Миша, и я пишу код!") # Английский текст print("I am learning Python") # С эмодзи print("🚀 Первый запуск — это победа!") # Умножаем строку: выведет имя 10 раз подряд print("Миша " * 10)

Секрет: Python понимает только прямые кавычки " и '. «Ёлочки» «» — это ошибка. Проверяй раскладку клавиатуры.

Красное в терминале — подсказка, а не ругань

Каждая ошибка говорит, что именно не так. Научись её читать

❌ Забыл закрыть скобку

print("Hello"

SyntaxError: '(' was never closed

→ Посчитай скобки: сколько открыл, столько закрой.

❌ Ёлочки вместо прямых кавычек

print(«Hello»)

SyntaxError: invalid character

→ Переключи раскладку на английскую.

❌ Опечатка в print

prin("Hello")

NameError: name 'prin' is not defined

→ Python не знает слова «prin». Читай свой код внимательно.

❌ Пробел перед командой

  print("Hello")

IndentationError

→ Python не любит лишние пробелы в начале строки.

Практика: запускаем локально

Задание 1 — открой GigaIDE и сделай свой первый запуск

  1. Открой GigaIDE Desktop с рабочего стола. Увидишь тёмное окно с боковой панелью.
  2. Создай новый файл — File → New File. Назови его hello.py. Расширение .py обязательно.
  3. Впечатай код ровно так:
    print("Hello, world")
  4. Сохрани файл — Ctrl + S. Увидишь галочку «сохранено».
  5. Нажми Run (зелёный треугольник ▶). Внизу должно появиться Hello, world.
  6. Сделай скриншот рабочего окна с надписью в терминале — это твой первый артефакт курса.

Практика: 3 версии приветствия

Задание 2 — поменяй текст внутри кавычек и запусти 3 раза

🇷🇺

По-русски

print("Привет, я Миша!")
🇬🇧

По-английски

print("Hi, I am Misha!")
🎉

С эмодзи

print("🚀 Я пишу код!")

Подумай: запусти код трижды — по одному варианту за раз. Что появляется в терминале каждый раз? Это ты «разговариваешь» с компьютером.

Практика: тот же код в облаке

Задание 3 — открой Colab и запусти print там же

  1. Открой colab.research.google.com в браузере
  2. Войди в Google-аккаунт (можно родительский)
  3. Нажми «Создать новый блокнот»
  4. В первой ячейке напиши print("Привет из облака!")
  5. Нажми ▶ слева от ячейки или Shift + Enter
  6. Сравни: где запускалось быстрее — в GigaIDE или в Colab?

🧠 Наблюдение: код совершенно одинаковый. Это значит: Python работает везде. Ты научишься писать его — и он будет жить в твоей программе на любом компьютере.

Мини-проект: «Визитка программиста»

Бонус для опережающих 🎁

Сделал основное задание — попробуй эти челленджи

🔁

Имя 10 раз подряд

print("Миша " * 10)

Видишь — строку можно умножать! Попробуй 100 раз.

📐

Зигзагом через табуляцию

print("Миша")
print("\tМиша")
print("\t\tМиша")

Символ \t — это табуляция, большой пробел. Сделай «лесенку» из имён.

Бонус-знакомство: ИИ-помощник 🤖

На этом уроке ИИ только показывается. На уроке 4 он начнёт помогать тебе с ошибками.

Пример запроса к GigaChat

Ты — детский преподаватель. Объясни, что такое Python,
10-летнему ребёнку тремя короткими предложениями,
с метафорой.

Правило курса: ИИ — это помощник, а не решатель. Он объясняет, подсказывает и спорит. Но код ты всё равно пишешь сам — иначе мозг не качается.
