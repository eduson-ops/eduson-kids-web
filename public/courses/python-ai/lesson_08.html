Новое на сегодня

Знакомься: .extend()

Метод extend() добавляет сразу несколько элементов — нужен список, кортеж или любой итерируемый объект.

Главное правило:
append — по одному.
extend — списком.
main.py
Python
diary = ["Кино"] # По одному diary.append("Пицца") # → ["Кино", "Пицца"] # Списком сразу diary.extend(["Футбол", "Книга"]) # → ["Кино", "Пицца", "Футбол", "Книга"] # А так — ОШИБКА: diary.append(["Футбол", "Книга"]) # → ["Кино", "Пицца", "Футбол", "Книга", # ["Футбол", "Книга"]] ← вложенный список!

Подключаем ИИ через extend()

У тебя уже есть 5 своих записей. Теперь добавь 2–3 готовых от ИИ одной строкой кода.

Важно: после extend список вырос — вызови sort() ещё раз, чтобы новые записи тоже встали на свои места.
Python
ai_entries = [ "[Футбол] Играл во дворе 40 минут", "[Книга] Прочитал 2 главы Гарри П.", "[Урок] Python у Алёши — круто" ] diary.extend(ai_entries) diary.sort() # ← ещё раз! print(f"Всего записей: {len(diary)}")

Подводные камни

5 типичных ошибок — и как их чинить

// 01

extend ждёт список

Нельзя extend("строка") — получишь буквы по одной. Нужен [список].

// 02

ИИ даёт markdown

Звёздочки и решётки в выводе? Попроси в промпте: «без markdown».

// 03

sort ломается

В списке только строки! Смешал строки и числа — ошибка типов.

// 04

SyntaxError в копипасте

Читай traceback: там номер строки. Ищи кривую кавычку или запятую.

// 05

Копирую всё из ИИ
