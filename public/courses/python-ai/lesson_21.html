// 02 · Метафора

Модуль — это учебник на полке

Представь библиотеку. На полках — тысячи учебников: математика, игры, время, таблицы, музыка. Python приходит к тебе с уже заполненной библиотекой.

import → «принёс учебник на стол»
. (точка) → «открыл на нужной странице»
from … import … → «вырвал одну страницу»
живой пример
Python
import math # открываем страницу "sqrt" в учебнике math print(math.sqrt(16)) # 4.0 # открываем страницу "pi" print(math.pi) # 3.14159...

// 04 · math — математика из коробки

Не пиши корень — он уже написан

Python
import math print(math.sqrt(16)) # 4.0 print(math.pi) # 3.141592653589793 print(math.floor(3.7)) # 3 (округление вниз) print(math.ceil(3.2)) # 4 (округление вверх) print(math.pow(2, 10)) # 1024.0

Что внутри

  • sqrt(x) — корень квадратный
  • pi — число π
  • floor / ceil — округление вниз / вверх
  • pow(a, b) — a в степени b
  • sin / cos / tan — тригонометрия
Фишка: math.pi — это не функция, это константа. Скобки не нужны.
// 05 · random — твой друг во всех играх

Случайность — двигатель игр

Python
import random # случайное целое от 1 до 6 (оба включительно) print(random.randint(1, 6)) # случайный выбор из списка герой = random.choice(["маг", "воин", "лучник"]) print(герой) # перемешать список колода = [1, 2, 3, 4, 5] random.shuffle(колода) print(колода) # [3, 1, 5, 2, 4] — каждый раз разный

Функции random

  • randint(a, b) — целое от a до b включительно
  • choice(список) — один случайный элемент
  • shuffle(список) — перемешать на месте
  • random() — число от 0.0 до 1.0
Опасно! randint(1, 6) включает 6. randrange(1, 6) — не включает 6. Легко запутаться.
// 06 · time — режиссёр пауз

Пауза — это драма

Python
import time print("Готов?") time.sleep(1) # пауза 1 секунда print("3") time.sleep(1) print("2") time.sleep(1) print("1") time.sleep(1) print("СТАРТ!")

Функции time

  • sleep(секунды) — остановить программу
  • time() — сколько секунд с 1970 года
  • можно дробные: sleep(0.3) — полсекунды
Ловушка: sleep(1000) — это 1000 секунд, почти 17 минут. Не миллисекунды! Только секунды.
// 07 · Две формы импорта

import vs from … import

форма 1 — весь учебник
import random

print(random.randint(1, 6))
print(random.choice(["a", "b"]))
  • нужен префикс random.
  • видно, откуда функция
  • удобно, если используешь много функций
форма 2 — одна страница
from random import randint, choice

print(randint(1, 6))
print(choice(["a", "b"]))
  • префикс не нужен
  • короче, но неочевидно откуда
  • удобно для 1–2 функций
Правило хорошего тона: все import — в самом верху файла. Сначала импорты, потом код.

// 10 · Практика 1

«Угадай число» — по-настоящему

Python
import random secret = random.randint(1, 100) print("Я загадал число от 1 до 100.") while True: guess = int(input("Твой вариант: ")) if guess < secret: print("Больше!") elif guess > secret: print("Меньше!") else: print("Победа!") break

Что тут новенького

  • import random — сверху файла
  • randint(1, 100) — целое от 1 до 100
  • секрет разный каждый запуск
  • while True + break — до победы
Идея: добавь счётчик попыток — кто угадал быстрее, тот герой дня.
// 11 · Практика 2

«Случайный герой»

Python
import random герои = ["маг", "воин", "лучник", "друид", "паладин"] выбор = random.choice(герои) print(f"Твой герой дня: {выбор}!")
Расширь: добавь оружие = random.choice([...]) и способность = random.choice([...]) — получишь уникального героя.

Что делает choice

Берёт один случайный элемент из любой последовательности: списка, кортежа, строки.

  • Работает со списком чисел
  • Работает со списком строк
  • Даже с буквами в строке: choice("abc")
// 12 · Практика 3

«Таймер обратного отсчёта»

Python
import time for i in range(5, 0, -1): print(i) time.sleep(1) print("СТАРТ!")

Что соединили

  • for — цикл, уже знаем
  • range(5, 0, -1) — от 5 до 1 с шагом −1
  • time.sleep(1) — пауза на секунду
  • в конце — команда «СТАРТ!»
Магия композиции: модули не заменяют циклы и условия — они склеиваются с ними. Это и есть настоящее программирование.
// 13 · Мини-проект

Мини-проект: «Броски костей»

Python
import random import math def roll_dice(n): """Бросает n костей, возвращает список.""" результаты = [] for _ in range(n): бросок = random.randint(1, 6) результаты.append(бросок) return результаты n = int(input("Сколько костей? ")) броски = roll_dice(n) print("Броски:", броски) print("Сумма:", sum(броски)) print("Среднее:", math.floor(sum(броски) / len(броски)))

Проверочный список

  • Два import — сверху
  • Функция с параметром n
  • Список результатов внутри
  • Возврат через return
  • Сумма через sum()
  • Среднее через math.floor
Бонус: найди максимум — max(броски). И минимум — min(броски). Это тоже встроенные функции, даже импорт не нужен.
// 14 · Типичные ошибки

Пять ловушек на пути

!

Забыли import

NameError: name 'random' is not defined — модуль не подключён.

A

import Random с большой буквы

Python регистрозависим. Правильно — только маленькими: random.

()

random(1, 6) без точки

TypeError: module is not callable. Нужно: random.randint(1, 6).

≠

Путают randint и randrange

randint(1, 6) — включает 6. randrange(1, 6) — не включает.

⏱

time.sleep(1000) — и тишина на 17 минут

sleep принимает секунды, не миллисекунды. Для полсекунды пиши sleep(0.5).
