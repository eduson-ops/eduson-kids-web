Определение

Строка — это бусы из символов

Текст в кавычках. Может быть любой: короткий, длинный, с пробелами, с эмодзи, пустой.

  • Одинарные: 'Привет'
  • Двойные: "Привет"
  • Пустая: ""
  • Тип данных называется str
Python
name = "Лена" city = 'Москва' emoji = "🐱🚀" empty = "" big = "Hello, World!" print(type(name)) # <class 'str'>

"Он \"Привет\""

Правило: выбирай те кавычки, которых нет в тексте — и будет красиво без обратных слэшей.
Функция len()

Линейка для текста

len() возвращает количество символов. Пробелы считаются. Эмодзи — обычно да.

Python
s = "Привет" print(len(s)) # 6 print(len("Hello!")) # 6 print(len("")) # 0 — пустая строка print(len(" ")) # 2 — два пробела print(len("🐱")) # 1
Лайфхак: если не уверен, сколько символов — спроси компьютер через len(). Он никогда не ошибается.

Оператор +

Склеиваем строки — конкатенация

+ между строками работает как клей. Результат — новая строка.

Важно: пробел сам не появится! Добавляй его руками: a + " " + b
Python
hi = "Привет" who = "мир" print(hi + ", " + who + "!") # Привет, мир! name = "Лена" greet = "Здравствуй, " + name print(greet) # Здравствуй, Лена
Ловушка!

"2" + "3" это не пять

Строки не умеют складываться как числа. Они просто встают рядом — получается "23".

Python
# Строки a = "2" b = "3" print(a + b) # "23" # Смешивать нельзя # print("5" + 3) TypeError
Python
# Если нужны числа: a = "2" b = "3" print(int(a) + int(b)) # 5 # Или наоборот print("5" + str(3)) # "53"
Оператор *

Повторяем через умножение

"Ха" * 5 — это не «два на пять», это пять копий подряд.

Python
print("Ха" * 5) # ХаХаХаХаХа print("-" * 20) # -------------------- print("🌟 " * 3) # 🌟 🌟 🌟 # Пробел между копиями: phrase = "Привет" n = 3 print((phrase + " ") * n) # Привет Привет Привет
Фишка Python

Строки нельзя менять на месте

Как лист, который уже напечатан на принтере. Букву стереть нельзя — но можно сделать новую строку.

Запомни термин: immutable — «неизменяемый».
Python
s = "привет" # Так нельзя! # s[0] = "П" # TypeError: 'str' object # does not support item # assignment # Так — можно: s = "П" + s[1:] print(s) # Привет

Ошибки-ловушки

Где чаще всего спотыкаются

🚫

IndexError: out of range

Обратился к s[100], а в строке 6 символов. Проверь через len().

⚠️

TypeError: str + int

"5" + 3 ломается. Реши: int("5") + 3 или "5" + str(3).

❝

Кавычки внутри кавычек

Чередуй '..."..."...' или экранируй \".

🔒

s[0] = "А" — нельзя

Строки неизменяемы. Создай новую: "А" + s[1:].
