Шаг 1 · 7 минут

Пароль: .lower().strip()

  • Задаём пароль
    Переменная password = "pythonMaster".
  • Просим ввод
    input() + сразу .strip() + .lower().
  • Сравниваем
    Важно: .lower() с обеих сторон!
game.py
Python
password = "pythonMaster" user_input = input("Введи пароль: ").strip().lower() if user_input == password.lower(): print("✅ Доступ разрешён") else: print("❌ Доступ запрещён")
⚡
Лайфхак
.strip() убирает случайные пробелы в начале и конце. .lower() переводит в нижний регистр. Комбо — защита от «PYTHONMASTER » и « PythonMaster».
Шаг 2 · 10 минут

Вложенное условие: выбор героя

game.py · шаг 2
Python
password = "pythonMaster" user_input = input("Введи пароль: ").strip().lower() if user_input == password.lower(): print("✅ Добро пожаловать, герой!") character = input("Выбери: воин / маг / лучник: ").strip().lower() print(f"Ты выбрал: {character}") else: print("❌ Доступ запрещён")

🔑 Главная идея

Второй input() — внутри ветки верного пароля. Неверный пароль = никакого выбора. Естественная иерархия.

🪜 Отступ решает

4 пробела — и код «вложен». Без отступа — он выполнится всегда, даже при неверном пароле. Это баг.

Ключевые фразы урока

.lower() с обеих сторон сравнения — защита от «Python», «python» и «PYTHON» одновременно.Фраза #2
.strip() — убираем случайные пробелы. Это вежливость к пользователю.Фраза #3
Вложенный if — пароль открывает дверь, за ней выбор героя. Естественная иерархия.Фраза #4
Ветка else — ловушка для неожиданного ввода. Всегда пиши.Фраза #5

Бонус для опережающих

Три попытки пароля без циклов

Идея: 3 вложенных if. Если первая попытка неверна — вторая. Если и вторая — третья. Иначе — блок.

Учебный момент
Это громоздко. На модуле про циклы решим то же самое тремя строчками.
