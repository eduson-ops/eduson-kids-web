Шаг 1 · 10 минут

Сначала — псевдокод

Прежде чем писать код, напиши план словами — прямо в файле, как комментарии.

Это называется псевдокод. Так делают все настоящие программисты: сначала думают, потом кодят.

Комментарий в Python начинается с # — Python его не читает, это текст для тебя.

Python
# helper_v1.py — Электронный помощник # 1. Спросить имя # 2. Спросить возраст (int) # 3. Спросить рост в см (int) # 4. Спросить вес в кг (int) # 5. Посчитать ИМТ = вес / (рост/100)**2 # 6. Округлить до 2 знаков # 7. Составить текст карточки # 8. Красиво вывести через print()
Шаг 2 · 15 минут

Собираем данные клиента

  • input() всегда возвращает строку.
  • Чтобы считать — оберни в int() или float().
  • Имя оставляем строкой — с ним не считаем.
4 input() 3 int() 1 str
Python
name = input("Как тебя зовут? ") age = int(input("Сколько тебе лет? ")) height = int(input("Твой рост в см? ")) weight = int(input("Твой вес в кг? "))
Шаг 2 · вычисления

Формула ИМТ

ИМТ — индекс массы тела. Его считают врачи, чтобы понять, в норме ли вес.

Формула: вес / (рост_в_метрах)²

Мы храним рост в сантиметрах, поэтому делим на 100 — и только потом возводим в квадрат.

Скобки = приоритет!
Python
# Переводим см в метры и возводим в квадрат height_m = height / 100 bmi = weight / (height_m ** 2) # Округляем до 2 знаков bmi = round(bmi, 2)
Шаг 3 · 10 минут

Красивый вывод через f-строки

f-строка — это строка, которая умеет «втягивать» значения переменных прямо в текст.

  • Ставим f перед кавычками.
  • Переменную пишем в { }.
  • \n — это перенос строки.
Python
card = ( f"=== Карточка клиента ===\n" f"Имя: {name}\n" f"Возраст: {age}\n" f"Рост: {height} см\n" f"Вес: {weight} кг\n" f"ИМТ: {bmi}\n" f"========================" ) print(card)
Итого · 30+ строк

Собираем программу целиком

# helper_v1.py — Электронный помощник ч.1

# --- 1. Сбор данных клиента ---
print("Здравствуйте! Заполним карточку клиента.\n")

name    = input("Как тебя зовут? ")
age     = int(input("Сколько тебе лет? "))
height  = int(input("Твой рост в см? "))
weight  = int(input("Твой вес в кг? "))

# --- 2. Вычисление ИМТ ---
height_m = height / 100
bmi = weight / (height_m ** 2)
bmi = round(bmi, 2)

# --- 3. Текст рекомендации (без условий — общий) ---
tip = "Поддерживай активность и пей воду!"

# --- 4. Вывод карточки ---
print()
print("=== Карточка клиента ===")
print(f"Имя:         {name}")
print(f"Возраст:     {age}")
print(f"Рост:        {height} см")
print(f"Вес:         {weight} кг")
print(f"ИМТ:         {bmi}")
print(f"Рекомендация: {tip}")
print("========================")
45 – 55 минут · Дебаг

А теперь я специально сломаю твой код

Учитель уберёт int() из одной строчки — и программа упадёт.

Твоя задача: не паниковать. Ошибка — это подсказка, а не провал.

Traceback читаем снизу вверх
Traceback (most recent call last): File "helper_v1.py", line 12, in <module> bmi = weight / (height_m ** 2) TypeError: unsupported operand type(s) for /: 'str' and 'float'
Навык разработчика

Как читать traceback

  1. Смотри на последнюю строку. Там тип ошибки и короткое объяснение.
  2. Найди номер строки в слове line 12 — вот где Python споткнулся.
  3. Посмотри на тип: TypeError = «неправильный тип данных».
  4. Спроси себя: «Что это за переменная? Я точно положил туда то, что нужно?»
  5. Если через 5 минут не нашёл — идём к ИИ-напарнику.
ИИ-напарник

Как правильно спросить YandexGPT

ИИ — не волшебник, который даёт готовый код. ИИ — напарник, который ОБЪЯСНЯЕТ.

  • Скажи, что ты учишься Python.
  • Вставь весь traceback — не урезай.
  • Попроси объяснить причину, а не дать готовое решение.
Markdown
Я учусь Python. Вот моя ошибка: Traceback (most recent call last): File "helper_v1.py", line 12, in <module> bmi = weight / (height_m ** 2) TypeError: unsupported operand type(s) for /: 'str' and 'float' Объясни по-детски, что случилось и на какой строке искать. Готовое решение НЕ давай.
Починка

Исправляем по объяснению ИИ

БЫЛО (сломано)

Python
weight = input("Твой вес в кг? ") # weight — это 'str', делить строку # на число нельзя → TypeError

СТАЛО

Python
weight = int(input("Твой вес в кг? ")) # теперь weight — это число, # arithmetic работает, ИМТ считается

Ты только что сделал то, что делают инженеры каждый день: прочитал ошибку → понял → починил.
