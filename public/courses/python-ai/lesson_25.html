Цель урока

Зачем нам новый контейнер?

Кроме списка у Python есть кортеж — запертый контейнер.

Используем его там, где данные не должны меняться:

координаты (x, y) дата рождения строка таблицы дни недели размер окна
🔒
Метафора

Список — шкаф, кортеж — сейф

📚 Список [ ]

  • 🔓 Открывается — можно менять
  • ➕ Добавляем .append()
  • ✏️ Меняем items[0] = "new"
  • 🗑️ Удаляем .remove()
  • Пример: инвентарь героя

🔒 Кортеж ( )

  • 🔐 На замке — менять нельзя
  • 📖 Только читаем days[0]
  • ⚠️ days[0] = ... → ошибка
  • 🏃 Быстрее и безопаснее
  • Пример: координаты точки

«Круглые скобки — замок закрыт. Квадратные — шкаф открыт.» 🔐

Синтаксис

Как создать кортеж

main.py
Python
# Координаты точки — (x, y) point = (10, 20) # Дни недели — 7 элементов days = ("пн", "вт", "ср", "чт", "пт", "сб", "вс") # Разные типы в одном кортеже — так можно! hero = ("Маг", 100, True, (15, 30)) # ⚠️ Одинарный кортеж — запятая ОБЯЗАТЕЛЬНА! single = (5,) # ← кортеж из одного элемента not_tup = (5) # ← это просто число 5 в скобках
⚠️
Запятая — секрет одинарного кортежа
(5) = число 5. (5,) = кортеж. Запятая говорит Python: «это контейнер!»
Чтение по индексу

Индексы — как у списка, с нуля

read_tuple.py
Python
days = ("пн", "вт", "ср", "чт", "пт", "сб", "вс") print(days[0]) # пн ← первый print(days[1]) # вт print(days[-1]) # вс ← последний print(days[2]) # ср # Длина print(len(days)) # 7 # Проверка «есть ли?» print("пт" in days) # True
📖

Читаем — можно

days[0] → "пн"

days[-1] → последний элемент

🚫

Менять — нельзя

days[0] = "суббота" → TypeError!

# Имя, HP, координаты — всё в одном кортеже hero = ("Маг", 100, (15, 30)) # Достаём по индексу print(hero[0]) # Маг print(hero[1]) # 100 print(hero[2]) # (15, 30) — вложенный кортеж print(hero[2][0]) # 15 ← X координата! print(hero[2][1]) # 30 ← Y координата # Перебираем цикл for for info in hero: print(info)

Задача: «Какой сегодня день?»

  1. Создать кортеж из 7 дней недели
  2. Спросить у пользователя номер дня (1–7)
  3. Вывести название дня
  4. Добавить проверку: если <1 или >7 — «неверный номер»
week.py
Python
days = ("пн", "вт", "ср", "чт", "пт", "сб", "вс") n = int(input("Номер дня (1–7): ")) if 1 <= n <= 7: print("Это:", days[n - 1]) else: print("Такого дня нет 🤔")
🎯
Секрет: n − 1
Пользователь считает с 1, а Python — с 0. Поэтому days[n - 1].

🎮 Бонус: кортеж в игре

В библиотеке PyGame размер окна — это всегда кортеж.

Почему? Потому что ширина и высота — фиксированы при создании и не должны случайно меняться.

Это «пакет настроек» — идеальный случай для кортежа.

game.py
Python
import pygame # Размер окна — кортеж (w, h) WINDOW = (800, 600) pygame.init() screen = pygame.display.set_mode(WINDOW) # Позиция героя — тоже кортеж! hero_pos = (400, 300) print("Размер:", WINDOW) # (800, 600) print("Ширина:", WINDOW[0]) # 800 print("Высота:", WINDOW[1]) # 600

Запомни навсегда

Три правила кортежа

🔒

1. Нельзя менять

Создал — всё, данные на замке. Это его суперсила.

( )

2. Круглые скобки

Создаётся через (), а не через [].

📖

3. Читается как список

Индексы с нуля, for, len(), in — всё работает.

Словарь урока

Новые термины

Кортеж (tuple)

Неизменяемый контейнер с элементами в круглых скобках.

Индекс

Номер элемента, с нуля. days[0] — первый день.

Неизменяемый

Создали — и больше не меняется. Как сейф.

TypeError

Ошибка, когда пробуем сделать то, что типу запрещено.

Распаковка

name, hp, pos = hero — быстрый способ достать значения.

Вложенность

Кортеж внутри кортежа. hero[2][0] — «папка в папке».
