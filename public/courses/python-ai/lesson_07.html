Идея одной картинкой

Метод — это кнопка на объекте

У каждого объекта в Python есть встроенные кнопки. Чтобы нажать кнопку — ставим точку и пишем имя метода.

name.upper()

объект · точка · кнопка

У списка тоже есть кнопки:

append() remove() pop() sort()

У строки — свои:

upper() lower() strip() replace()
Словарь урока

Что значит каждое слово

метод

Методы строк

Очищаем и меняем текст

Python
s = " Привет, Мир! " print(s.strip()) # "Привет, Мир!" print(s.upper()) # " ПРИВЕТ, МИР! " print(s.lower()) # " привет, мир! " print(s.replace("Мир", "Python")) # " Привет, Python! "

Четыре встроенные кнопки

  • strip() — убрал пробелы
  • upper() — сделал ВСЁ БОЛЬШИМ
  • lower() — сделал всё маленьким
  • replace(a, b) — заменил одно на другое

Заметь: строка s сама не меняется. Метод возвращает новую строку.

Профи-приём

Цепочка методов: одна строка — три действия

Python
s = " Привет, МИР! " result = s.strip().lower().replace("мир", "python") print(result) # "привет, python!"
" Привет, МИР! " → .strip() "Привет, МИР!" → .lower() "привет, мир!" → .replace() "привет, python!"

Каждый метод возвращает новую строку — следующий метод работает с результатом предыдущего.

Методы списков

Списки меняются сами

Python
lst = [3, 1, 2] lst.append(10) # lst = [3, 1, 2, 10] lst.sort() # lst = [1, 2, 3, 10] lst.pop() # вернул 10, lst = [1, 2, 3] lst.remove(2) # lst = [1, 3] print(lst) # [1, 3]

Кнопки списка

  • append(x) — добавил в конец
  • sort() — отсортировал
  • pop() — вытащил последний
  • remove(x) — удалил первое x

Обрати внимание: lst меняется! Присваивать ничего не нужно.

Главная разница

Строки возвращают. Списки меняют.

Строки · неизменяемые

Python
s = "hi" s.upper() # "HI" print(s) # "hi" ← не изменилось! s = s.upper() # так правильно print(s) # "HI"

Метод возвращает новую строку. Присваивай обратно.

Списки · изменяемые

Python
lst = [3, 1, 2] lst.sort() print(lst) # [1, 2, 3] ← изменился! # НЕ надо так: lst = lst.sort() print(lst) # None 💥

Метод меняет список. Не присваивай обратно.

Типичная ошибка № 1

Почему print(lst.sort()) выводит None?

Python
lst = [3, 1, 2] print(lst.sort()) # None ← не то, что ждали # Правильно: lst.sort() print(lst) # [1, 2, 3]

Sort — это уборщик

Он убрал в списке — и молча ушёл. Ничего не вернул.

Если хочешь увидеть результат — печатай сам список после вызова sort().

✕
Правило: методы, которые меняют список (sort, append, remove) возвращают None. Не присваивай их результат.

Мини-проект · 10 минут

«Редактор заметок»

Что делает программа

  1. Спрашивает заметку у пользователя
  2. Очищает пробелы по краям (.strip())
  3. Делает первое слово ПРОПИСНЫМИ (через срез и .upper())
  4. Заменяет слово «плохо» на «отлично» (.replace())
  5. Добавляет заметку в список notes (.append())
  6. Сортирует список и выводит его (.sort())

Это четыре метода строк и два метода списков в одной программе. Почти бейдж!

Референс-решение

Код проекта: 15 строк

Python
notes = [] note = input("Заметка: ").strip() # Первое слово — ЗАГЛАВНЫМИ space = note.find(" ") if space == -1: note = note.upper() else: first = note[:space].upper() rest = note[space:] note = first + rest # Заменяем "плохо" на "отлично" note = note.replace("плохо", "отлично") notes.append(note) notes.sort() print(notes)
Типичные сложности

Четыре грабли, на которые все наступают

1
print(lst.sort()) выводит None
sort ничего не возвращает. Сначала вызвать, потом печатать lst.
2
s.upper() не меняет s
Строки неизменяемы. Нужно s = s.upper().
3
remove(x) ломается, если x нет
ValueError. Ловим ошибку на уроке про условия.
4
Путают append и extend
append — один элемент, extend — много. Разберём на уроке 8.
Проверь себя

Три вопроса на понимание

Q1

Метод строки меняет саму строку?

Нет. Возвращает новую.

Q2

Что вернёт lst.sort()?

None. Сортирует на месте.

Q3

Что делает pop()?

Убирает последний и возвращает его.

Бонус для быстрых

.split() и .join() — мост между строкой и списком

Python
text = "я учу python каждый день" words = text.split(" ") # ["я", "учу", "python", "каждый", "день"] print(f"В предложении {len(words)} слов") # В предложении 5 слов back = " ".join(words) # "я учу python каждый день"

Челлендж: спроси предложение, выведи сколько в нём слов. Используй split() и len().
