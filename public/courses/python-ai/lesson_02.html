Первые переменные — вместе

Открываем GigaIDE или Colab и печатаем строчку за строчкой:

age = 10
name = "Миша"
print(type(age), type(name))

age = age + 1          # переопределение
print(age ** 2, 10 // 3, 10 % 3)
Ожидаемый вывод: <class 'int'> <class 'str'> · 121 3 1
🧑‍💻 Правила имён

Как правильно называть переменные

✅
Хорошо age, my_age, player_hp, year_born, days_in_year
❌
Плохо мой возраст, 2age, my age, class
  • Английские буквы, цифры и нижнее подчёркивание _.
  • Начинать не с цифры.
  • Без пробелов: вместо пробела — _.
  • Имя должно говорить, что внутри: не x, а player_hp.
➗ Арифметика Python

Семь операторов

+Сложение5 + 3 → 8
−Вычитание5 − 3 → 2
*Умножение5 * 3 → 15
/Деление (с дробью)10 / 3 → 3.333…
//Целочисленное деление10 // 3 → 3
%Остаток от деления10 % 3 → 1
**Возведение в степень2 ** 5 → 32
🍬 Метафора конфет

Зачем три разных деления?

🍭
10 / 3«Разделим честно, в том числе по крошкам». Ответ: 3.333…
🍫
10 // 3«Каждому по целой конфете». Ответ: 3 (каждому).
🎯
10 % 3«Сколько осталось». Ответ: 1 конфета лишняя.

Проверь на себе: 15 // 4, 15 % 4. Что вышло?

🔬 Функция type()

Рентген для коробки

Python сам определяет тип — но ты можешь спросить у него напрямую:

Python
x = 42 print(type(x)) # <class 'int'> x = 3.14 print(type(x)) # <class 'float'> x = "привет" print(type(x)) # <class 'str'>
  • int — целое число (без точки)
  • float — дробное (с точкой)
  • str — строка (в кавычках)
Заметь: одна и та же переменная x может менять тип — коробка гибкая!
📏 Лайфхак для больших чисел

Нижнее подчёркивание _ — для тебя

Большие числа в Python можно разделять подчёркиваниями. Для программы это всё ещё одно число, а ты не запутаешься, где тысячи и миллионы.

1_000 · тысяча 1_000_000 · миллион 1_000_000_000 · миллиард
Python
gold = 1_000_000 population = 8_100_000_000 print(gold) # 1000000 print(population) # 8100000000 # Python игнорирует _ в числах print(1_000 == 1000) # True

Мини-физика: скорость

Скорость = путь ÷ время. Это формула из учебника — и она уже работает в Python.

  • Задай distance и time.
  • Вычисли speed.
  • Выведи красиво через print(...) с запятой.
Python
distance = 100 time = 20 speed = distance / time print("Скорость:", speed, "м/с") # → Скорость: 5.0 м/с
🎮 Мини-проект · 10 мин

«Паспорт персонажа»

Придумай героя из любимой игры. Заполни на него «паспорт» через переменные и посчитай пару характеристик.

  • Имя, уровень, здоровье, атака, золото (1_000_000).
  • Посчитай «урон в секунду» и «цена 5 таких героев».
  • Покажи скрин в чат.
Python
name = "Астра" level = 42 hp = 1200 attack = 85 gold = 1_000_000 dps = attack * 2 # 2 удара в секунду total_price = gold * 5 print(name, "ур.", level) print("HP:", hp, " ATK:", attack) print("DPS:", dps) print("5 таких стоят:", total_price)

⚠️ Ловушки и выходы

Где часто спотыкаются

1
Путает = и === — положить в коробку. == — сравнить. Сравнение встретим позже.
2
Имена кириллицейТехнически можно. В профи-коде — нельзя. Договариваемся на английский.
3
Забыл кавычки у строкиОшибка NameError: ... is not defined. Читаем вместе вслух.
4
Путает / и /// — с запятой. // — целая часть. % — остаток. Метафора конфет.
5
a - = 5 — пробелыПробелы между - и = ломают оператор. Правильно: a -= 5.
✨
Всё это — нормаКаждый программист так ошибается. Главное — читать ошибку и не бросать.
