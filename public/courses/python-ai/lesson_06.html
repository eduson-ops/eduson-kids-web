Пока у нас была одна переменная на одно значение. Но задачи растут: оценки за месяц, список песен, ответы на анкету. Писать ball1, ball2, ball3... — неудобно и некрасиво.

Метафора урока

Список — это подъезд с квартирами

Представь подъезд. В каждой квартире кто-то живёт. У каждой квартиры есть номер. В Python эти «номера» называются индексами и начинаются с нуля.

fruits = ["яблоко", "банан", "груша", "киви"]
0"яблоко"
1"банан"
2"груша"
3"киви"
Запомни: индексы в Python всегда начинаются с 0. Первый элемент — [0], не [1]. Это главная ловушка новичков.

Синтаксис

Как создать список

Квадратные скобки. Элементы через запятую. Всё. Это и есть список.

Python
# Список строк fruits = ["яблоко", "банан", "груша"] # Список чисел scores = [5, 4, 3, 5, 2] # Пустой список basket = [] # Разные типы — тоже ок mixed = [10, "Вася", 3.14, True]
  • Открывающая скобка [
  • Элементы через ,
  • Закрывающая скобка ]
  • Строки — в кавычках
  • Числа — без кавычек
  • Пустой список — просто []
Лайфхак: переменную для списка называем во множественном числе: fruits, scores, names.
Правило индексов

Индекс = номер ячейки

Чтобы достать элемент, пишем имя списка и индекс в квадратных скобках: fruits[0].

songs = ["Imagine", "Bohemian", "Hotel California", "Yesterday", "Let It Be"]
0"Imagine"
1"Bohemian"
2"Hotel C."
3"Yesterday"
4"Let It Be"
Python
songs[0] # "Imagine" songs[1] # "Bohemian" songs[4] # "Let It Be"
Важно: если в списке 5 элементов, индексы идут от 0 до 4. Длина — 5, но последний индекс — 4. Почему? Потому что считаем с нуля.
Суперсила Python

Индексы с конца

Python умеет считать с конца списка через минус. -1 — последний элемент, -2 — предпоследний. Очень удобно.

week = ["Пн", "Вт", "Ср", "Чт", "Пт", "Сб", "Вс"]
0 / -7Пн
1 / -6Вт
2 / -5Ср
3 / -4Чт
4 / -3Пт
5 / -2Сб
6 / -1Вс
Python
week[-1] # "Вс" — последний день week[-2] # "Сб" — предпоследний week[0] # "Пн" — первый
Изменяемость

Список можно менять на месте

В отличие от строки, элементы списка можно переписать. Просто обращаемся по индексу и присваиваем новое значение.

Python
fruits = ["яблоко", "банан", "груша"] print(fruits) # ['яблоко', 'банан', 'груша'] fruits[1] = "вишня" print(fruits) # ['яблоко', 'вишня', 'груша']
  • fruits[1] = "вишня" — меняем элемент
  • Остальные элементы не трогаем
  • Длина списка не меняется
  • Новую переменную не создаём
Не путай:
fruits[0] = "x" — меняем элемент.
fruits = [1,2,3] — создаём новый список.
Гибкость Python

В одном списке — разные типы

Строки, числа, булевы, даже другие списки. Python не против. Это часто используют, чтобы хранить карточку объекта.

Python
# Карточка игрока player = ["Вася", 12, 3.14, True] name = player[0] # "Вася" level = player[1] # 12 score = player[2] # 3.14 alive = player[3] # True print(f"{name}, ур. {level}")

str — строка

"Вася" — текст в кавычках

int — целое число

12 — уровень, количество

float — дробное

3.14 — средний балл

bool — да/нет

True / False — живой?

Инструмент

Сколько элементов? len()

Встроенная функция len() возвращает количество элементов в списке.

Python
songs = ["Imagine", "Yesterday", "Let It Be"] print(len(songs)) # 3 # Первый элемент: songs[0] # Последний элемент: songs[len(songs) - 1] # или songs[-1]
  • len(songs) = количество элементов
  • Если длина 3 — индексы 0, 1, 2
  • Последний индекс = len(songs) - 1
  • Или проще: songs[-1]
  • Длина пустого списка — 0
Типичные ошибки

Что пойдёт не так

IndexError

Обращение к индексу, которого нет.

lst = [1, 2, 3]
lst[5]  # IndexError!

Фикс: проверяй len(lst).

( ) вместо [ ]

Круглые — это кортеж, не список.

lst = (1, 2, 3)  # кортеж
lst = [1, 2, 3]  # список

Фикс: для списка — квадратные.

Забыл запятую

SyntaxError при создании списка.

lst = [1 2 3]      # ошибка
lst = [1, 2, 3]    # ок

Фикс: запятая после каждого.

input() ≠ список

Ввод даёт строку, не список.

lst = input()  # это строка!

lst = [input(), input(), input()]

Фикс: собирай список из отдельных input().

Проект: Дневник оценок

Программа спрашивает 5 оценок, хранит их в списке и выводит каждую по индексу.

Python
print("=== Дневник оценок ===") grades = [ int(input("Оценка 1: ")), int(input("Оценка 2: ")), int(input("Оценка 3: ")), int(input("Оценка 4: ")), int(input("Оценка 5: ")), ] print("---") print(f"День 1: {grades[0]}") print(f"День 2: {grades[1]}") print(f"День 3: {grades[2]}") print(f"День 4: {grades[3]}") print(f"День 5: {grades[4]}") print(f"Все оценки: {grades}")
  • 5 вопросов через input()
  • Оборачиваем в int() — чтобы было число
  • Всё складываем в один список grades
  • Выводим каждую через f-строку
  • Показываем весь список в конце
Результат: готовая программа-дневник. Скриншот — в чат преподавателю.
Средний балл? Научимся через 2 урока, когда пройдём циклы.
Бонус для опережающих

Список внутри списка

В Python элементом списка может быть другой список. Получается таблица. Так делают поле для крестиков-ноликов или карту в игре.

Python
board = [ ["X", "O", "X"], ["O", "X", "O"], ["X", "O", "X"], ] # Весь ряд print(board[0]) # ['X', 'O', 'X'] # Конкретная ячейка print(board[1][1]) # X — центр print(board[0][2]) # X — верх-право # Изменяем board[2][0] = "O"
X | O | X
—————
O | X | O
—————
X | O | X

Первый индекс — ряд. Второй — колонка. board[1][1] — центр поля.
