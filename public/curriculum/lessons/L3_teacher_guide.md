## Цели урока

По завершении ученик умеет:

1. Объяснить модель событий: «когда X — сделай Y».
2. Использовать хотя бы 3 события: `@старт`, `@столкновение`, `@нажата_клавиша`.
3. Создать монетку (объект), которая исчезает при касании игрока.
4. Добавить зону финиша с `@столкновение(финиш)`.
5. Объяснить разницу между событием и обычной командой.

---

## Подготовка

- Открыть `L3_events_collisions.html` в браузере.
- Открыть KubiK Editor с проектом из Урока 2 (есть движение персонажа).
- Добавить в сцену вручную: 1 монету, 1 шип, 1 зону финиша.

## Пошаговый план урока

### Блок 1: Проблема — статичная игра (5 мин)

Вопрос: «Что происходит в вашей игре, когда персонаж касается монеты?» → Ничего. «Как сделать так, чтобы монета исчезала?» → Нужно событие.

Аналогия: «Событие — как звонок телефона. Пока не позвонят (касание) — ничего не происходит. Позвонили — запустился обработчик (функция).»

### Блок 2: Три вида событий (10 мин)

Пройди по слайдам с примерами блоков:

1. **`@старт`** — запускается один раз при старте игры. «Как утреннее приветствие».
2. **`@столкновение(объект)`** — запускается при касании. «Как дверной звонок».
3. **`@нажата_клавиша(пробел)`** — при нажатии. «Как кнопка лифта».

Для каждого: покажи в редакторе, добавь в скрипт, запусти.

### Блок 3: Монетки-практика (15 мин)

**Минимум (9–11 лет):**
```kubik
@столкновение(монета):
    удалить(монета)
    сказать "Монета!"
```

**Расширенный (12+):**
```kubik
мир счёт = 0

@старт:
    сказать "Собирай монеты!"

@столкновение(монета):
    счёт += 10
    удалить(монета)
    сказать(соединить("Счёт: ", текст(счёт)))

@столкновение(шип):
    сказать "Упс! Шип!"
```

### Блок 4: Зона финиша (7 мин)

```kubik
@столкновение(финиш):
    сказать "Победа!"
    установить_небо("вечер")
```

Обсуди: «Почему мы используем разные имена объектов (`монета`, `шип`, `финиш`)? Как игра понимает, с чем столкнулся игрок?» — по имени тега объекта.

### Блок 5: Тест (5 мин)

Два quiz-вопроса из слайдов. Проверить: дети понимают, что событие не надо явно вызывать — оно запускается само.

### Блок 6: Итог (3 мин)

Каждый ребёнок называет вслух: «В моей игре есть событие __ которое делает __».

---

## Частые ошибки

| Ошибка | Причина | Решение |
|---|---|---|
| Событие не срабатывает | Имя объекта не совпадает | Проверить имя тега в Properties |
| `удалить` убирает не то | Передан неверный аргумент | `удалить(монета)` — тот объект, с которым столкнулись |
| Счёт не меняется | `мир счёт = 0` не объявлен | Объявить перед первым событием |
